# 라디오 드롭다운 목록

쿼리 조건의 기본 방법은 사용자가 쿼리하기 위해 매개 변수 값을 수동으로 입력해야 하는 텍스트 상자입니다. 실제 비즈니스에서 사용자는 매개 변수 값의 전체 이름을 입력하거나 기억할 수 없는 값을 잘 모르는 경우가 높으므로 매개 변수의 드롭다운 목록 기능이 필요합니다.

매개 변수의 드롭다운 목록 기능을 사용하면 수동으로 입력할 필요 없이 드롭다운 목록에서 직접 옵션을 선택하여 쿼리를 구현할 수 있습니다.



예를 들어 데이터 필터링 처리를 추가하지 않은 간단한 보고서를 예로 들어 테이블 구성 요소에 데이터 필터를 추가하여 범주 이름으로 데이터를 필터링할 수 있습니다.

![](https://help.grapecity.com.cn/download/thumbnails/56537608/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1615948012000\&api=v2)  테이블을 추가하고 필드를 바인딩합니다.

![](https://help.grapecity.com.cn/download/thumbnails/56537608/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1615948012000\&api=v2) 보고서 매개 변수를 추가합니다. 보고서 디자이너 오른쪽에 있는 데이터 바인딩 패널을 열고 다음 그림과 같이 보고서 매개 변수 영역의 추가를 클릭합니다.

보고서 매개 변수 목록에 새 항목인 보고서 매개 변수 1이 나타나고 해당 항목을 클릭하면 보고서 매개 변수에 대한 자세한 설정 정보가 패널에 표시됩니다.

사용 가능한 의 데이터 세트 이름 드롭다운 상자에서 위에 정의된 제품 범주 데이터 집합을 선택하고, 데이터 필드 드롭다운 상자에서 범주 이름을 선택하고, 레이블 필드 드롭다운 상자에서 다음 그림과 같이 범주 이름을 선택합니다.

![](<../../../.gitbook/assets/image (1126).png>)

여기서 데이터 필드의 값은 데이터 집합 SQL 문에 전달된 실제 매개 변수 값으로 사용됩니다. 레이블 필드의 값은 드롭다운 목록 상자에 표시되는 문자열입니다.

이렇게 하면 사용 가능한 데이터에 대한 쿼리 매개 변수가 설정되고 보고서 보기 페이지의 매개 변수 표시줄에 드롭다운 목록 상자로 표시되어 사용자가 선택할 수 있습니다.

![](https://help.grapecity.com.cn/download/thumbnails/56537608/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1615949722000\&api=v2)  데이터 필터를 추가합니다. 전체 테이블 구성 요소를 선택하여 속성 설정 패널을 엽니다. \[데이터 필터] 항목 뒤에 있는 \[편집]을 클릭합니다.

필터를 구성합니다. 여기서 필터는 범주 이름 필드로 구성됩니다(오른쪽을 클릭하고 드롭다운 목록에서 범주 이름을 선택합니다). 데이터는 보고서 매개 변수로 구성됩니다(오른쪽을 클릭하고 드롭다운 목록에서 범주 이름을 선택합니다).![](https://help.grapecity.com.cn/download/thumbnails/56537608/image2021-10-13_19-6-45.png?version=1\&modificationDate=1635241194000\&api=v2)![](https://help.grapecity.com.cn/download/thumbnails/56537608/image2021-10-13_19-6-38.png?version=1\&modificationDate=1635241194000\&api=v2)

![](<../../../.gitbook/assets/image (1165).png>)

![](https://help.grapecity.com.cn/download/thumbnails/56537608/%E6%AD%A5%E9%AA%A45.png?version=1\&modificationDate=1615949842000\&api=v2)  보고서를 미리 봅니다. 매개 변수 패널에서 드롭다운 버튼을 클릭하면 드롭다운 목록 스타일의 쿼리 매개 변수가 표시됩니다. 드롭다운 상자에서 쿼리 조건을 선택하기만 하면(키워드를 입력하여 자동으로 검색할 수 있습니다) 범주별로 제품 정보를 쿼리할 수 있습니다.
