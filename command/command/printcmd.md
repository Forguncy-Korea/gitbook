# 인쇄 명령 - 웹 페이지 인쇄 명령

웹 페이지 인쇄 명령을 사용하여 현재 페이지를 인쇄합니다.

"인쇄 요소 숨기기"를 선택하면 버튼, 하이퍼링크 또는 그림과 같은 인쇄 명령을 실행하는 요소가 인쇄되지 않고 선택되지 않은 경우 인쇄 요소가 인쇄됩니다.

{% hint style="info" %}
* 인쇄할 때 테이블의 스크롤 막대와 선택한 효과(선택된 색상 및 테두리)가 자동으로 숨겨집니다.
* 인쇄 시 셀의 드롭다운 및 위쪽 및 아래쪽 버튼이 자동으로 숨겨집니다.
{% endhint %}

## 웹 페이지 인쇄 명령&#x20;

페이지에서 셀 범위를 선택하고 버튼 설정하고 명령을 인쇄 명령으로 설정합니다.

실행 후 페이지에서 인쇄 버튼을 클릭하면 현재 페이지가 인쇄됩니다.

예를 들어 기본 제공 직원 정보 관리 시스템을 열고 직원 관리 홈 페이지에서 셀 범위를 선택하여 버튼으**로** 설정하고 명령을 인쇄 명령으로 편집합니다.&#x20;
