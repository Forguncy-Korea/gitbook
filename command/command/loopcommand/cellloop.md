# 셀에 따라 루프

루프 명령에서 순환이 셀별로 반복되는 경우 기준 셀 수인 기준 셀 수를 지정해야 합니다.

하위 명령에서 수식이 상대 참조를 사용하는 경우 루프 중에 수식이 참조하는 셀이 기준 셀로 오프셋됩니다.

![](<../../../.gitbook/assets/image (2105).png>)

## 셀에 따라 루프&#x20;

셀별로 순환할 때 기준 셀을 지정하는 방법에는 여러 가지가 있습니다.

* "A1, A2, A3"과 같은 영어 쉼표로 구분된 셀 참조;
* 셀 범위 참조(예: "A1:C1")
* 셀 및 셀 범위에 대한 혼합 참조(예: "A1, A2:A3")
* A1, A2 및 A3과 같은 쉼표로 구분된 셀 이름은 각각 a, b, c이며 "a,b,c"를 사용하여 기준 셀을 지정할 수 있습니다.

셀별로 순환할 때 하위 명령에 리스트뷰 템플릿 행의 셀을 참조하는 수식이 있는 경우 절대 참조를 사용합니다. 루프 중에 참조 오프셋으로 인해 테이블에서 선택한 행의 값을 얻을 수 없습니다.

예를 들어 주문 목록 페이지의 루프 버튼에서 명령을 반복 명령으로 설정하고, 셀 루프로 루프를 선택하고, 셀을 A1:C1로 선택한 후 Enter 키를 누르면 셀 수가 아래에 자동으로 계산됩니다.&#x20;

![](<../../../.gitbook/assets/image (2180).png>)

다음 그림과 같이 루프의 하위 명령을 데이터 테이블 작업으로 설정하고 다음 그림과 같이 데이터 테이블 작업을 추가합니다.

![](<../../../.gitbook/assets/image (1418).png>)

실행 후 주문 목록 페이지의 "루프" 버튼을 클릭하면 테이블 아래에 세 개의 새 레코드가 추가된 것을 볼 수 있습니다.

![](<../../../.gitbook/assets/image (1803).png>)
