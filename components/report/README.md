# 보고서

이 보고서는 현대 기업 정보화에 필수적인 통계 분석 도구이며, 주로 기업 내 상대적으로 고정 된 일간지, 판매 주간 보고서, 재무 월별 보고서 및 주요 데이터의 통계 분석과 같은 보다 상세한 데이터 프리젠 테이션 분석을 실현하는 데 사용됩니다.

보고서 모듈을 통해 개발자는 다양한 보고서를 쉽게 작성할 수 있습니다. 최종 사용자는 이러한 보고서를 브라우저에서 직접 볼 수 있으며 클릭 한 번으로 PDF 파일로 내보낼 수 있어 인쇄 및 정리에 편리합니다.

다음은 포건시 보고서 모듈을 사용하여 빌드된 일반적으로 사용되는 보고서 유형입니다.

* Excel 양식(재무제표, 회계제표, 통계제표)
* Word 문서 클래스(품질 검사 보고서, 테스트 결과 보고서, 불규칙 테이블)
* 대시보드 관리 Kanban(대형 화면 조종석)
* 그룹 보고서, 열 보고서, 트리 구조 다층 통계 보고서
* 모든 종류의 인쇄 문서 및 보고서

이 장에서는 포건시 보고서의 사용에 대해 설명합니다.
