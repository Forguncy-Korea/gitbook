# 도메인 이름 설정

포건시는 전자 메일 전송을 지원하며 회사 인트라넷 사용자의 경우 일반적으로 전자 메일 콘텐츠의 URL에 액세스하기 위해 인트라넷 DNS가 필요합니다. 포건시는 도메인 이름 설정을 지원하며, 이를 통해 사용자 전자 메일의 URL이 인트라넷에서 포건시 서버로 이동합니다.

## 도메인 이름 설정&#x20;

서버관리자에서 앱의 도메인이름을 설정합니다.&#x20;

![](https://help.grapecity.com.cn/download/thumbnails/72363211/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092698000\&api=v2)  서버관리의 앱 목록에서 앱을 선택하고 앱 이름을 클릭하여 앱의 일반 설정 페이지로 이동합니다.

![](<../../.gitbook/assets/image (876).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72363211/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092698000\&api=v2)  도메인 이름 설정 영역에서 https://www.domain.com:8080 같은 새 도메인 이름을 입력하고 설정 저장을 클릭합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72363211/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092698000\&api=v2)  도메인 이름을 설정한 후 앱을 다시 시작하여 적용합니다. 확인을 클릭하여 앱을 다시 시작합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72363211/%E6%AD%A5%E9%AA%A44.png?version=1\&modificationDate=1648092697000\&api=v2) 앱이 성공적으로 다시 시작되면 앱 목록에서 앱을 클릭한 후 앱으로 이동하면 앱의 URL이 설정된 도메인 이름이 된다는 것을 알 수 있습니다.![](https://help.grapecity.com.cn/download/thumbnails/72363211/image2020-2-28_12-0-21.png?version=1\&modificationDate=1648092697000\&api=v2)
