# 페이지 디자인

웹페이지는 웹페이지의 인터페이스이다. 웹 정보 시스템은 일련의 웹 페이지로 구성됩니다. 각 웹 페이지에는 관련 데이터와 비즈니스 프로세스가 포함되어 있으므로 먼저 웹 페이지가 있어야 합니다. 포건시에서는 한 페이지가 하나의 웹페이지에 해당합니다.

### 페이지 <a href="#ye-mian" id="ye-mian"></a>

포건시의 페이지에는 일반 페이지, 모바일 페이지, 마스터 페이지, 내장 페이지가 포함됩니다.

**일반 페이지**

일반 페이지는 웹 페이지에 해당합니다. 공통 페이지를 설정하고 관리할 수 있습니다.

페이지 관리: 폴더를 생성하고 폴더에 관련 페이지 그룹을 배치할 수 있습니다.

**모바일 페이지**

모바일 페이지는 모바일 브라우저 접속에 사용되는 페이지를 의미합니다.

**마스터 페이지**

마스터 페이지는 여러 일반 페이지에서 공유할 수 있는 페이지의 공유 부분을 디자인하는 데 사용됩니다. 예를 들어 탐색 모음은 마스터 페이지에서 디자인할 수 있습니다.

마스터 페이지는 일반 셀 영역과 페이지 공간의 두 부분으로 나뉩니다. 일반 셀 영역에는 네비게이션 바와 같은 공유 부분을 디자인할 수 있으며, 페이지 자리 표시자 영역은 일반 페이지의 표시를 전환하는 데 사용됩니다.

**내장 페이지**

포건시에서는 포건시 템플릿의 일부 기본 페이지를 보거나 내장 페이지 및 내장 모바일 페이지를 포함하는 새로운 빈 프로젝트를 생성할 수 있습니다.

내장된 페이지는 포건시에서 특수 기능을 지정하고 특정 시간에 표시됩니다. 시스템에 401이 발생하면 자동으로 401 페이지로 이동합니다. 사용자가 로그인해야 하면 자동으로 로그인 페이지로 이동합니다.

내장 페이지의 인터페이스와 모양을 수정할 수 있지만 로그인 페이지의 사용자 이름, 비밀번호, 로그인 버튼과 같은 특수 기능이 있는 텍스트 상자, 버튼 또는 하이퍼링크는 삭제할 수 없습니다.
