# Excel 명령 - 페이지를 Excel로 내보내기

페이지를 Excel로 내보내기 명령을 사용하여 전체 페이지의 내용을 Excel로 내보낼 수 있습니다.

페이지를 Excel로 내보내는 것은 내보내기 명령에 속합니다.\

## 페이지를 Excel로 내보내기&#x20;

Excel로 페이지를 내보내는 설정 지침은 표에 나와 있습니다.

<table><thead><tr><th width="338.14865284728575">설정 </th><th>설명 </th></tr></thead><tbody><tr><td>페이지 내보내기 </td><td>앱에서 페이지를 선택하여 Excel로 내보냅니다.</td></tr><tr><td>내보낸 Excel파일의 테이블 오버플로 모드 </td><td><p></p><p>테이블의 데이터 행 수가 테이블의 행 수보다 크면 테이블 데이터의 내보내기 모드를 설정할 수 있습니다.</p><ul><li>테이블 행 수 자동 증가: 테이블의 행 수를 늘리면 모든 데이터가 표시됩니다.</li><li>Sheet 수를 늘리고 여러 Sheet에 더 많은 테이블 데이터를 표시합니다. 예를 들어 테이블에는 10개의 행이 있고 테이블에는 20개의 행이 있으며 11~20개의 행은 두 번째 sheet로 내보내집니다.</li></ul></td></tr><tr><td>Excel 파일 이름 </td><td><p>내보낸 Excel의 파일 이름을 페이지 이름, 페이지 제목 또는 기타 이름과 일치하도록 설정할 수 있으며 다른 이름을 선택할 때 고정 값, 지정된 셀 또는 수식을 입력할 수 있습니다. </p><ul><li>이름을 설정할 때 이름에 !#%&#x26;*+=|\:'&#x3C;>?,/특수 문자를 포함할 수 없습니다.</li><li>이름이 수식인 경우 수식 계산 오류는 오류 메시지를 표시합니다</li></ul></td></tr><tr><td>고급설정</td><td><p></p><ul><li>내보내기 중 페이지의 수식 처리 모드: 내보내기 페이지가 현재 페이지인 경우 이 옵션을 체크하면 페이지의 수식이 내보내지지 않고, 수식으로 계산된 값만 내보내집니다. 다른 페이지를 내보낼 때 수식과 값이 모두 내보내집니다. 내보낸 수식은 Excel의 수식 편집 표시줄에서 볼 수 있습니다.</li><li>내보낸 파일을 암호화하고 암호화 비밀번호 설정: 내보낸 파일을 암호화하려면 다음 텍스트 상자에 암호화 비밀번호를 설정하세요. 비밀번호는 직접 설정하거나 페이지의 셀을 참조하도록 설정할 수 있습니다.</li></ul></td></tr></tbody></table>

예를 들어 주문정 페이지에서 \[내보내기] 버튼을 설정하는 명령은 \[페이지를 Excel로 내보내기], 페이지는 기본 \[현재 페이지], 테이블 데이터 내보내기 모드는 \[엑셀에 리스트뷰 모든 내용을 표시], Excel 파일 이름은 \[기타 이름]을 선택하고 M4 셀을 지정합니다.&#x20;

![](<../../.gitbook/assets/image (720).png>)

실행 후 주문 테이블\_목록 페이지에서 테이블의 데이터가 모두 표시되지 않습니다. \[페이지 내보내기] 단추를 클릭하면 내보낸 Excel의 이름이 M4셀의 값인 \[주문 목록]이며, 여기서 테이블의 행 수가 자동으로 증가하고 테이블의 모든 데이터가 표시되고 페이지의 모든 내용이 Excel로 내보내집니다.

![](<../../.gitbook/assets/image (557).png>)

{% hint style="info" %}
* **마스터 페이지에서 페이지를 Excel로 내보내기 명령을 실행하면 페이지 자리 표시자의 하위 페이지만 내보내집니다.**
* **페이지에 테이블이 있는 경우 테이블이 내보낸 테이블에 영향을 주는 데이터를 요청 시 로드할 수 있는지 여부입니다.**
  * **테이블에 로드 레코드 행 수가 설정되어 있고 요청 시 로드가 활성화되지 않은 경우. 이 시점에서 테이블에는 로드된 레코드 행 수에 대한 일부 데이터만 표시할 수 있으며 표시된 데이터만 내보낼 수 있습니다.**
  * **테이블에 로드 레코드 행 수가 설정되어 있고 요청 시 로드가 활성화된 경우. 테이블에는 모든 데이터가 표시되고 내보낼 때 내보낼 수 있습니다**
{% endhint %}
