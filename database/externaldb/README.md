# 외부 데이터베이스 연결

포건시의 데이터베이스 유형에는 두 가지 유형이 있습니다.

* 기본 제공 데이터베이스: 유형 SQLite는 빠른 처리 능력을 갖춘 경량 데이터베이스입니다.
* 타사 데이터베이스: SQL Server, Oracle, MySQL 및 MariaDB 대규모 데이터베이스를 기본으로 지원하며 PostgreSQL, Access 및 기타 데이터베이스에 대한 ODBC 연결을 지원합니다.

포건시는 외부 데이터베이스를 지원하며 SQL Server, Oracle, MySQL 및 MariaDB를 기본으로 지원합니다. 외부 데이터베이스에서 가져온 테이블은 내장된 데이터 테이블과 마찬가지로 바인딩 및 업데이트를 지원합니다.

외부 데이터베이스의 최신 데이터가 포건시에 표시되고 라이브 그리드 앱에서 업데이트된 데이터가 아웃리치 데이터베이스에 동기화됩니다.

## 사용 시나리오&#x20;

*   **기존에 있던 데이터베이스를 사용**

    기존 데이터베이스를 직접 사용하려면 앱이 최신 데이터를 참조할 수 있는 외부 데이터베이스를 사용할 수 있습니다. 예를 들어 회사의 영업 데이터베이스에 연결하고, 포건시에 필요한 판매 데이터를 추출하고, 분석을 위한 응용 프로그램 시스템을 구축합니다.
* **2TB 이상의 데이터를 처리하는 빅 데이터**

&#x20;     포건시의 모든 내부 테이블에서 처리할 수 있는 데이터의 양은 2TB이며 더 많은 양의 데이터를 처리하려면 아웃리치 데이터베이스를 사용하여 처리해야 합니다.  &#x20;

*   **높은 동시 쓰기**&#x20;

    매우 높은 동시 쓰기가 필요한 경우 외부 데이터베이스를 사용해야 합니다. 한 순간(몇 초 이내)에 100명이 동시에 양식을 제출합니다. 포간의 내부 데이터베이스는 동시 쓰기가 높을 때 성능이 저하되지만 하루 중 다른 시간에 내부 데이터베이스를 커밋해도 성능이 저하되지 않습니다.
