# 서버단 명령의 반환값

서버  명령을 정의할 때 비즈니스 요구에 따라 반환 값을 정의할 수 있습니다. 반환 값에는 기본적으로 반환 코드와 반환 정보가 포함되며, 더 많은 반환 값을 정의할 수 있습니다.

반환 값은 문자열, 숫자 등의 단순 형식뿐만 아니라 배열, 개체 등의 복잡한 값 형식도 지원합니다.
