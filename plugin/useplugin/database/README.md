# 데이터베이스 연결 플러그인

포건시 V10에서는 아키텍쳐 레벨에서 데이터베이스  플러그인을 지원하는  기능을 도입하였습니다. 이 기능이 도입된다는 것은 과거에는 포건시가 새로운 데이터베이스를 지원하기 위해서는 공식적인 개발과 테스트를 기다린 후 다음 메이저 버전에서 출시해야 한다는 것을 의미합니다. 하지만 이제 새로운 데이터베이스를 지원하는 기능을 구현하려면  새로운 플러그인을 설정하기만 하면 됩니다. 특정 데이터베이스에 문제가 발견되면 다음 주요 버전을 기다릴 필요 없이 메시어스 포럼에서   플러그인만 설치하면 됩니다. 더욱 흥미로운 점은 미래에는 개발자가 필요한 데이터베이스 플러그인을 직접 개발하여 포건시에서 직접 사용할 수도 있다는 것입니다
