# 메뉴&#x20;

메뉴는 전체 웹사이트의 분류 및 구조를 보다 명확하게 할 수 있고, 찾기 및 사용이 더 편리합니다. 메뉴 유형 셀을 사용하여 페이지를 전환합니다. 메뉴 플러그인은 모바일 페이지 및 모바일 마스터 페이지에서 사용할 수 없습니다.

### 사용 방법&#x20;

1. 플러그인을 다운로드합니다.
2. Forguncy Builder에서 설치하고 Forguncy Builder를 다시 실행합니다.
3. 셀 범위를 선택하고 해당 셀 유형을 메뉴로 설정합니다.&#x20;

<figure><img src="../../../../.gitbook/assets/image (2078).png" alt=""><figcaption></figcaption></figure>

4\. 속성 설정 영역의 "셀 설정" 탭에서 메뉴를 설정하며, 설정 설명은 표 1을 참조한다.



| 설정         | 설명                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 메뉴 항목 편집   | <p>1. 설정 메뉴의 항목, 다단계 메뉴를 설정할 수 있으며, <img src="https://gcdn.grapecity.com.cn/data/attachment/forum/201807/18/112735ac1iwu7wiw6ozgzc.png" alt=""> 메뉴 항목을 증가, 감소, 위로 이동 및 아래로 이동합니다.<br>2. 메뉴 항목의 명령, 즉 이 메뉴 항목을 클릭했을 때 실행할 명령을 설정합니다.<br>3. 이름, 아이콘, 알림 등 메뉴의 속성을 설정합니다. 딸깍 하는 소리 <img src="https://gcdn.grapecity.com.cn/data/attachment/forum/201807/18/112813hwkfw250zifwk56g.png" alt=""> 셀을 선택하고 실행하면 이 수준의 메뉴에 알림 수가 표시되며 그 값은 선택한 셀의 값입니다.<br>4. 메뉴 항목의 권한을 설정하면 이 메뉴의 역할에 접근할 수 있습니다.<br></p> |
| 방향         | 가로 및 세로 방향을 포함한 메뉴의 레이아웃을 설정합니다.                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| 메뉴 스타일 편집  | 색상, 글꼴, 글꼴 크기, 아이콘 크기 등 메뉴의 스타일을 설정합니다.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
