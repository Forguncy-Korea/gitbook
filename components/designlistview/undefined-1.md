# 요약 행 표시하기

리스트뷰 요약 행을 지원합니다. 요약 행을 켜면 평균, 개수, 최대값, 최소값, 합계 또는 일반 텍스트를 포함하여 리스트뷰의 각 열을 요약할 수 있으며 요약 행의 셀 스타일을 설정할 수 있습니다.



리스트뷰를 선택하고 마우스 오른쪽 버튼을 클릭한 다음 마우스 오른쪽 버튼 클릭 메뉴에서 리스트뷰 옵션 설정을 선택하여 리스트뷰 설정 대화 상자를 표시합니다.

리스트뷰 설정의 모양 탭에서 요약 행을 선택하면 리스트뷰의 요약 행이 열립니다. 기본 요약 행은 리스트뷰 맨 아래에 있으며 "요약행을 위쪽 첫 번째 줄에 표시하기"를 선택하면 요약 행이 테이블의 맨 아래에 표시됩니다.



![](<../../.gitbook/assets/image (1386).png>)

리스트뷰를 선택한 후 리본 메뉴 모음에서 리스트뷰 도구-디자인을 선택하고 스타일 옵션 영역에서 요약 행 표시를  선택하여 리스트뷰의 요약 행을 열 수도 있습니다.

## 요약 행 설정하기&#x20;

요약 행을 연 후 테이블의 각 열에 대해 요약 행을 개별적으로 설정할 수 있습니다.

리스트뷰에서 열을 선택하고 셀 설정에서 \[요약 행 세부 구성]을 클릭하면 요약 행을 설정하는 대화 상자가 나타납니다.

![](<../../.gitbook/assets/image (930).png>)

요약 행의 수식 유형 및 스타일을 설정할 수 있습니다.

수식 유형에는 일반 텍스트, 평균, 개수, 최대값, 최소값 및 합계가 포함됩니다.

예를 들어 리스트뷰에서 구매제품 열과 금 열을 선택하고 요약 행의 수식 유형을 일반 텍스트로 설정하고 텍스트를 제품으로 설정합니다.

![](<../../.gitbook/assets/image (1913).png>)

수량 및 금액 열을 선택하여 요약 행의 공식 유형을 합계로 설정합니다.

![](<../../.gitbook/assets/image (1207).png>)

모든 열의 스타일을 지정하고 채우기 색상을 설정합니다.

![](<../../.gitbook/assets/image (779).png>)

실행되면 요약 행이 리스트뷰 아래에 표시됩니다.

![](<../../.gitbook/assets/image (679).png>)
