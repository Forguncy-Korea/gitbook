# 보고서 내보내기 명령

이 섹션에서는 보고서 명령의 보고서 내보내기 명령에 대해 설명합니다.

보고서 내보내기 명령을 사용하여 브라우저에서 보고서를 내보내고 명령의 보고서 매개변수 및 데이터 소스 매개변수에 매개변수 값을 전달할 수 있습니다.

보고서 내보내기 명령은 서버 명령에서도 지원됩니다.



예를 들어 페이지에서 "보고서 내보내기" 버튼을 설정하고 버튼의 명령을 편집한 다음 해당 명령을 보고서 내보내기 명령으로 선택합니다.

보고서 내보내기 명령을 설정합니다.

* 내보내기할  보고서: 내보낼 보고서를 선택하고 여러 보고서를 한 번에 내보낼 수 있습니다.
* 파일 이름: 내보낸 보고서의 파일 이름을 설정합니다.
* 파라미터 목록: "파라미터 이름"의 드롭다운 버튼을 클릭하면 모든 데이터 소스 매개변수와 보고서 매개변수가 나열됩니다.
