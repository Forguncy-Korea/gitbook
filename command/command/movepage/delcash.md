# 고급설정 - 캐시 제거

이동한 페이지가 내부 페이지, 즉 앱 내의 페이지인 경우 내부 페이지 간에 이동할 때 고급 설정을 수행할 수 있습니다.

고급 설정에서 페이지 쿼리 및 정렬에 대한 캐시를 제거할지 여부를 설정할 수 있습니다.
