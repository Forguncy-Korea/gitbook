# 페이지 명령 - 메세지 보여주기

페이지에 메시지 상자를 표시하고 특정 정보를 표시하는 텍스트를 표시합니다. 메시지가 경고 상자로 나타납니다.

메시지 텍스트를 사용자 지정하거나, 입력 상자에 직접 텍스트 내용을 입력하거나, 수식 사용을 선택하거나, 수식을 입력하거나, 페이지의 셀을 지정할 수 있습니다.

![](<../../.gitbook/assets/image (1891).png>)

## 메시지 보여주기 설정&#x20;

* 메시지 텍스트를 직접 입력합니다

메시지 텍스트 입력 상자에서 비즈니스 요구 사항에 따라 프롬프트 텍스트를 직접 입력할 수 있습니다.

예를 들어 메시지 텍스트를 팝업 메시지 상자로 설정합니다.

![](<../../.gitbook/assets/image (451).png>)

메시 상자는 브라우저마다 다르게 스타일이 지정됩니다. 예를 들어 크롬 다음과 같이 표시됩니다.

![](<../../.gitbook/assets/image (835).png>)



* "수식 사용"을 선택합니다.

메시지 텍스트를 선택한 후 수식 사용을 선택하거나, 수식 입력 상자에 수식을 입력하거나, 페이지의 셀을 지정합니다.

예를 들어 수식 사용을 선택한 후 페이지의 텍스트 상자 셀을 선택합니다

![](<../../.gitbook/assets/image (1604).png>)

실행 후 텍스트 상자에 텍스트를 입력한 다음 단추를 클릭하면 텍스트 상자에 입력한 텍스트인 메시지 상자가 나타납니다.

![](<../../.gitbook/assets/image (1890).png>)
