# 키워드

키워드라고 하는 %로 시작하고 %로 끝나는 이동형에는 특별한 의미가 있는 문자열이 있습니다.

포건시는 현재 로그인한 사용자의 일부 속성 정보와 같은 응용 프로그램의 특정 데이터를 쉽게 얻을 수 있는 일련의 키워드를 제공합니다.



키워드는 페이지에 정보를 표시하고, 쿼리 명령과 함께 특정 레코드를 쿼리하고, 조건부 명령과 함께 특정 프로세스를 제어하고, 데이터 테이블 작업 명령과 함께 특정 값을 데이터베이스로 업데이트하는 등의 작업을 수행하는 데 사용할 수 있습니다.

키워드의 유연한 사용은 응용 프로그램의 기능을 보다 유연하게 만들고 권한 제어를 보다 세밀하게 할 수 있습니다.<br>

### 키워드 삽입&#x20;

\[삽입>키워드]를 선택하고 키워드 목록에서 키워드를 클릭하여 페이지에 키워드를 삽입할 수 있습니다.

<figure><img src="../.gitbook/assets/image (1690).png" alt=""><figcaption></figcaption></figure>

그 중 사용자 이름, 사용자 전체 이름, 이메일 주소, 역할은 사용자의 기본 속성이고, 직위는 사용자 정의 속성이며, 회사, 사업부, 부서는 조직 수준입니다.

예를 들어 페이지에 키워드를 삽입하고 실행 후 로그인한 사용자의 페이지 이름과 속성 정보를 페이지에 표시할 수 있습니다.
