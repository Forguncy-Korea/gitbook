# 저장소 경로 설정

포건시의 서버관리자에서 앱 파일, 감사 로그, 업로드 파일 및 동시 사용자 로그를 비롯한 다양한 파일에 대한 저장소 경로를 설정할 수 있습니다.

## 저장소 경로 설정 저장&#x20;

포건시의 서버관리자에서 \[설정-> 저장소 경로설정]을 선택하여 저장소 경로 페이지로 이동합니다.

<figure><img src="../../.gitbook/assets/image (146).png" alt=""><figcaption></figcaption></figure>

* 애플리케이션 기본 경로: 게시된 앱을 저장하며 기본적으로 \[C:₩Users₩Public₩Documents₩ForguncyServer]에 저장됩니다.
* 로그 경로: 데이터 로그는 데이터베이스에서 수행하는 모든 작업을 기록합니다. 데이터 로깅을 사용하면 오류 분석, 동작 분석, 보안 감사 등을 수행하여 데이터 실행을 효과적으로 얻을 수 있습니다. 기본값은 \[C:\Users\사용자 이름\AppData\Local\Temp]에 저장됩니다. 게시된 앱 로그는 \[C:\Windows\Temp]에 저장됩니다.
* 업로드 기본 경로: 이미지 업로드 또는 첨부 파일을 통해 포건시에 내장된 데이터베이스의 이미지 및 파일에 업로드되며 기본적으로 \[C:\Users\Public\Documents\ForguncyServer\\_응용 프로그램 이름_\Upload] 디렉토리에 저장됩니다.
* 동시 사용자 로그 경로 : 동시 사용자의 로그를 저장하며 기본적으로 \[C:\WINDOWS\TEMP\Forguncy\ForguncyConcurrencyAccessLog]에 저장됩니다.
* 프로젝트 파일 경로 : 배포 중에 서버에 릴리스된 후 프로젝트 파일의 저장 디렉터리입니다. 기본적으로 "C:\Users\Public\Documents\ForguncyDocumentBackup"에 저장됩니다.

설정이 완료되면 설정 저장을 클릭합니다.



**첨부 파일은 경로 설정을 저장합니다**

최종 사용자 업로드 파일의 루트에 설정된 모든 앱에 대한 첨부 파일의 저장소 경로 또는 각 앱에 대해 별도의 첨부 파일 저장소 경로를 설정할 수 있습니다.

서버관리자에서 앱 관리를 클릭하고 첨부 파일 저장소 경로를 설정할 앱을 선택하고 고급 설정에서 첨부 파일 저장소 경로를 설정하고 설정이 완료되면 설정 저장을 클릭합니다.
