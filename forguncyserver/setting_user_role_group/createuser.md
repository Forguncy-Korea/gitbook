# 사용자 생성

사용자를 만드는 방법에 대해 설명합니다.

사용자 계정 관리 플랫폼을 개발할 때 사용자를 만드는 것은 사용자가 공식적으로 출시된 후와 동일한 방식으로 사용자를 만드는 것입니다.

<figure><img src="../../.gitbook/assets/image (2427).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
사용자를 추가하려면 서버 및 사용자 인증 라이선스를 추가해야 합니다.
{% endhint %}

&#x20;사용자-사용자 관리 내장 영역에서 사용자를 클릭하여 모든 사용자를 표시합니다. Administrator는 기본적으로 추가된 사용자이며 삭제할 수 없습니다.

## 사용자 추가하기&#x20;

사용자 목록 위에서 사용자 추가를 클릭하고 사용자 추가를 선택하여 사용자를 추가합니다.

### Forguncy 인증 사용자 추가&#x20;

사용자 이름, 전체 이름, 암호, 확인 암호, 이메일 주소를 포함한 사용자 정보를 설정합니다.

<figure><img src="../../.gitbook/assets/image (2428).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
* 사용자 이름, 암호, 확인 암호 및 이메일 주소가 필요합니다.
* 기본 암호 정책은 문자, 숫자 또는 기호여야 하는 6자 이상이어야 합니다. 암호 정책을 참조하여 암호 정책을 변경할 수 있습니다.
{% endhint %}

### Windows 인증 사용자 추가&#x20;

검색 상자에 검색 키워드를 입력하고 Enter 키를 클릭하거나 눌러 검색하고 사용자를 선택한 다음 확인을 클릭하여 사용자를 성공적으로 추가할 수 있습니다.![](https://help.grapecity.com.cn/download/thumbnails/72356598/image2018-2-7_11-52-55.png?version=1\&modificationDate=1648092594000\&api=v2)

## 사용자 정의 속성&#x20;

전화, 주소 등과 같은 사용자 고유의 속성을 사용자 지정 속성으로 직접 설정할 수 있습니다.

사용자 관리 영역에서 사용자 지정 속성을 선택하면 인터페이스에 모든 사용자 지정 속성이 나열됩니다.

\[새 특성 추가]를 클릭하여 팝업 대화 상자에서 속성 이름과 범주를 설정합니다. 사용자 지정 또는 문자 속성의 사용자 지정 속성을 설정할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (2430).png" alt=""><figcaption></figcaption></figure>

예를 들어 제목이라는 속성을 추가하고, 속성 범주가 텍스트인 사용자 지정 속성을 추가하고, 추가가 완료되면 사용자를 추가하고, 사용자 인증 모드를 "Forguncy 인증 사용자 추가"로 선택합니다.

제목 속성이 맨 아래에 추가되고 사용자를 추가할 때 필수가 아닌 항목을 채울 수 있습니다.
