# HTML 편집기 (HtmlRichEditor)

HtmlRichEditor 플러그인을 사용하여 형식, 글꼴, 글꼴 크기, 배경색, 표, 등 을 사용할 수 있습니다.

{% hint style="info" %}
HTML Editor에서 저장하는 데이터 형식은 HTML 태그입니다.&#x20;
{% endhint %}

### 사용방법&#x20;

1. 플러그인을 다운로드합니다.
2. Forguncy Builder에서 설치하고 Forguncy Builder를 다시 실행합니다.
3. 셀 범위를 선택하고 해당 셀 유형을 HTML 편집기로 설정합니다.
