# 최신 버전을 가져오기

프로젝트를 편집하고 변경 내용을 커밋한 후 다른 사용자가 체크 아웃하여 편집을 계속할 수 있습니다. 그러나 프로젝트를 체크 아웃하려면 최신 버전의 프로젝트를 가져와야 합니다.

이 섹션에서는 최신 버전을 가져오는 프로젝트에 대해 설명합니다.

## 최신 버전 가져오기&#x20;

사용자가 변경 내용을 수정하고 체크 인한 후 다른 사용자가 동일한 모듈을 편집할 때  체크 아웃하기 전에 최신 버전의 프로젝트를 가져오라는 메시지가 표시됩니다.



예를 클릭하면 사용자가 체크 아웃하고 편집할 수 있을 때까지 최신 버전의 프로젝트가 자동으로 가져옵니다.

최신 버전의 엔지니어링 파일을 수동으로 가져올 수도 있습니다.

리본 메뉴 모음에서 \[공동작업-> 최신 버전 가져오기]를 선택하고 드롭다운 버튼을 클릭하여 최신 버전을 가져오거나 최신 버전으로 강제 동기화하도록 선택합니다.

* 서버에서 최신 버전 가져오기: 모든 사용자가 제출한 수정 사항을 포함하여 최신 버전의 엔지니어링 파일을 가져옵니다.
* 서버를 원격 동기화하여 사용하기: 최신 버전을 가져오고 로컬로 커밋되지 않은 모든 변경 내용이 취소됩니다.
