# Snowflake 데이터베이스

포건시는 아키텍처 레벨에서 데이터베이스 플러그인을 지원하는 기능을 도입합니다. 이 플러그인을 사용하면 포건시에서 Snowflake 데이터베이스를 연결하여 사용할 수 있습니다.&#x20;



### 플러그인 다운로드&#x20;
