# 입력

모바일 화면에서 텍스트를 입력할 수 있습니다.&#x20;

기본 제공 텍스트 상자와 비교하였을 경우 아래와 같은 이점과 제약사항이 있습니다.&#x20;

### 이점

1. 현대적인 UI 디자인
2. 제한 최대 길이 지원 및 최대 길이 설정 시 입력 단어 제한 표시
3. 접두사 및 접미사 아이콘 지원
4. 모든 텍스트를 지우는 지우기 아이콘 지원
5. 두 가지 확장 방법 포커스 및 텍스트 선택

### 제약사항&#x20;

1. 목록 보기에 셀 유형 추가를 지원하지 않음
2. 고유한 데이터 유효성 검사를 지원하지 않습니다.
3. 포커스를 받을 때 모든 텍스트 선택을 지원하지 않음
4. 스타일 템플릿을 지원하지 않음
5. Excel 형식을 지원하지 않음&#x20;
