# 데이터 명령 - 데이터 간 이동하기

데이터 간 이동하기를 사용하면 지정된 데이터 테이블의 데이터에 대한 특정 이동 수행할 수 있습니다. 페이지에 테이블 관련 테이블이 있는 경우 테이블의 현재 포커스가 있는 행, 즉 현재 행이 이동합니다.

![](<../../.gitbook/assets/image (586).png>)

## 데이터 간 이동하기&#x20;

데이터 테이블을 선택한 후 데이터 테이블의 레코드에 대한 특정 이동을 수행할 수 있습니다.

| 설정         | 설명                                                 |
| ---------- | -------------------------------------------------- |
| 데이터테이블 선택  | 이동을 기록할 데이터 테이블을 선택합니다.                            |
| 대상 데이터 선택  | 레코드를 첫 번째 데이, 이전 데이터 다음 데이터 는 마지막 데이터 이동하도록 설정합니다. |

예를 들어 주문 목록의 데이터 이동 버튼에서 데이터 이동에 대한 명령을 설정하고 데이터 테이블을 주문 테이블로 선택하고 데이터가 마지막 데이터로 이동하도록 설정합니다.&#x20;

![](<../../.gitbook/assets/image (2082).png>)

실행 후 \[데이터 이동] 버튼을  클릭하면 다음 그림과 같이 현재 행이 마지막 레코드로 이동합니다.
