# 셀유형 작동 명령&#x20;

다른 셀 유형에는 다른 작업이 있습니다. 셀 유형  작동  명령을 사용하면 대상 셀 유형을 선택할 때 셀의 사용 가능한 작업이 나열되고 대상 셀에서 지원하는 작업을 볼 수 있으며 실제 비즈니스 요구에 따라 사용할 작업을 선택할 수 있습니다.



셀 유형 작동명령을 사용할 때 대상 셀을 선택해야 합니다. 대상 셀에 대한 셀 유형이 설정되지 않은 경우 \
"선택된 셀없유형에는  가능한는작업이 없습니다"   라는메시지가 표시됩니다.

<figure><img src="../../.gitbook/assets/image (467).png" alt=""><figcaption></figcaption></figure>

예를 들어 대상 셀을 F2로 설정하고 셀 유형을 콤보 상자로 설정하면 대상 셀을 선택하면 해당 셀에서 수행할 수 있는 작업이 "작동"에 나열됩니다.

일부 셀 유형은 매개변수를 설정하고 값을 반환하고 후속 명령에서 사용할 수 있는 변수에 반환 값을 저장할 수도 있습니다.

설정이 완료된 후 셀을 동작시키는 명령을 실행하기 위해 페이지를 실행하면 설정된 동작이 실행됩니다.&#x20;
