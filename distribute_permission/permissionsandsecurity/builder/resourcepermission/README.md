# 리소스권한

리소스 권한에는 셀 권한, 페이지 권한, 서버단 명령 권한, 서버단 알림 권한, 보고서 권한 및 프로세스 권한이 포함됩니다.

* [셀 권한](cellpermission.md) : 셀의 가용성 권한, 가시성 권한, 편집 가능 권한을 설정하여 각 역할의 사용자가 액세스/편집할 수 있는 요소를 정확하게 제어할 수 있습니다.
* [페이지 권한](pagepermission.md) : 각 페이지에 허용되는 역할을 설정합니다.
* [서버 명령 권한](servercommandpermission.md) : 서버 명령을 호출할 수 있는 역할을 설정합니다.
* [서버 알림 권한](servernotificationpermission.md#she-zhi-fu-wu-duan-tong-zhi-quan-xian) : 설정구독 가능이 서버 알림의 역할입니다.
* [보고서 권한](reportpermission.md) : 보고서를 열 수 있는 권한과 보고서를 편집할 수 있는 역할을 설정합니다.
