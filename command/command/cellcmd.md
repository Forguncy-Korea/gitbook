# 셀 명령 - 셀 속성과 내용 변경하기

셀 등록 정보 설정 명령을 사용하여 대상 셀에 대한 속성 및 값을 설정합니다.

![](<../../.gitbook/assets/image (1304).png>)

| 설정     | 설명                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 대상 셀   | <p>속성을 설정할 셀입니다. 셀 수식 또는 셀 이름을 직접 입력하거나 아이콘을 눌러 셀을 선택할 수 있습니다.<img src="https://help.grapecity.com.cn/download/thumbnails/72354393/8-1-10-13_2.png?version=1&#x26;modificationDate=1648092562000&#x26;api=v2" alt=""></p><p>여러 셀에 대해 동일한 셀 속성과 값을 설정하려면 여러 대상 셀을 선택할 수 있습니다.</p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| 속성 유형  | <p></p><p>다음 셀 등록 정보를 설정할 수 있습니다.</p><ul><li>대상 셀에 값을 입력: 셀에 고정된 값을 설정하고 직접 입력할 수 있습니다.</li><li>보이기(Visible)설정: 셀이 표시되는지 여부를 설정합니다.</li><li>활성화(Enable)설정: 셀을 사용하지 않도록 설정할지 여부를 설정합니다.</li><li>읽기/쓰기 여부 설정: 셀을 읽기 전용 또는 읽기/쓰기로 설정합니다.</li><li>배경 색상: 셀에 배경색을 설정하거나, 16진수 색상 코드를 직접 입력하거나, 특정 색상을 선택할 수 있습니다.</li><li>글자 색상: 셀에 글자색을 설정하고 16진수 색상 코드를 직접 입력하거나 특정 색상을 선택할 수 있습니다.</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 설정 값   | <p></p><p>셀에 값을 설정하여 직접 입력하거나 아이콘을 클릭하여 셀 또는 셀 범위를 선택합니다.<img src="https://help.grapecity.com.cn/download/thumbnails/72354393/8-1-10-13_2.png?version=1&#x26;modificationDate=1648092562000&#x26;api=v2" alt=""></p><p>값을 클릭하여 선택할 수도 있습니다. 셀 속성 설정은 다르며 선택할 수 있는 값은 다릅니다.<img src="https://help.grapecity.com.cn/download/thumbnails/72354393/image2018-2-1_15-6-22.png?version=1&#x26;modificationDate=1648092562000&#x26;api=v2" alt=""></p><ul><li>셀 등록 정보가 [값]으로 설정된 경우 값은 [%Null%], [%AutoGeneratedID%], [%CurrentUer.UserName%], [%CurrentUer%], [%CurrentUer.FullName%], [%CurrentUer.Email%],<br>[%CurrentUer.Role%]를 선택합니다.</li><li>셀 속성이 [표시/보이지 않음]으로 설정된 경우 값은 [표시] 또는 [보이지 않음]을 선택할 수 있습니다.</li><li>셀 속성이 사용/사용 안 함으로 설정된 경우 값은 [사용] 또는 [사용 안 함]을 선택할 수 있습니다.</li><li>셀 속성이 읽기 전용/읽기/쓰기로 설정된 경우 값은 읽기 전용 또는 읽기/쓰기를 선택할 수 있습니다.</li><li>셀 속성이 배경색으로 설정된 경우 값은 테마 색상, 표준 색상 및 기타 색상을 선택할 수 있습니다.</li><li>셀 속성이 전경색으로 설정된 경우 값은 테마 색상, 표준 색상 및 기타 색상을 선택할 수 있습니다.</li></ul> |

실행 후 셀 속성 설정 버튼을 클릭하면 페이지의 주문 관리 셀 배경이 노란색으로 바뀝니다.

<figure><img src="../../.gitbook/assets/image (2313).png" alt=""><figcaption></figcaption></figure>

실행 후 "색상변경" 버튼을 클릭하면 페이지의 "주문 관리" 셀 배경이 노란색으로 변합니다.
