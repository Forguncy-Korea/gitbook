# CellRange 인터페이스

셀 범위의 위치 정보입니다. getDesignerRangeInfo 메서드에 사용됩니다.



| Row         | number | 이 셀 범위의 시작 행 인덱스입니다. |
| ----------- | ------ | -------------------- |
| Column      | number | 이 셀 범위의 시작 열 인덱스입니다. |
| RowCount    | number | 이 셀 범위의 행 수입니다.      |
| ColumnCount | number | 이 셀 범위의 열 수입니다.      |
