# 텍스트 상자

셀을 텍스트 상자 유형으로 설정하고 실행 후 텍스트 상자에 텍스트를 입력할 수 있습니다. 텍스트 상자는 그림에 나와 있습니다.

<figure><img src="../../.gitbook/assets/image (1531).png" alt=""><figcaption></figcaption></figure>

## 텍스트 상자 설정&#x20;

텍스트 상자를 선택하고 속성 설정 영역에서 셀 설정 탭을 선택합니다.

<table><thead><tr><th width="218">항목 </th><th>설명 </th></tr></thead><tbody><tr><td>명령편집 </td><td><p>값이 변경될 때 실행되는 명령을 설정합니다. 명령은 포커스가 텍스트 상자를 벗어나거나 Enter 키를 누를 때만 커밋됩니다.</p><p>명령은 <a href="../../command/command/">명령</a>을 참조하십시오.</p></td></tr><tr><td>데이터검증</td><td>페이지 측 유효성 검사에서 텍스트 상자의 데이터 <a href="../../pages/verification_data/page/">유효성 검</a>사를 설정합니다.</td></tr><tr><td>기본값</td><td><p>기본 표시 텍스트를 설정합니다(예: 기본값 A001). 텍스트 상자에는 A001이 표시됩니다.</p><p>수식을 입력하면 수식 결과가 기본적으로 표시됩니다.</p></td></tr><tr><td>암호표시</td><td>이 옵션을 선택하면  입력한 문자가 *로 표시됩니다.</td></tr><tr><td>고유한 연동 테이블 필드</td><td>이 옵션을 선택하면 데이터 테이블이 업데이트되거나 추가될 때 텍스트 상자에 데이터베이스에 이미 있는 값과 동일한 값을 입력할 수 없습니다. 텍스트 상자에 데이터 바인딩이 설정된 경우에만 작동합니다. (텍스트 상자가 리스트뷰에 있는 경우 이 옵션이 없습니다.)</td></tr><tr><td>워터마크</td><td>위의 그림과 같이 셀에 내용이 없을 때 표시되는 프롬프트 텍스트는 워터마크가 "텍스트를 입력하십시오"입니다.</td></tr><tr><td>아이콘 선택</td><td><p></p><p>텍스트 상자에 대한 아이콘을 설정합니다.</p><p>[이미 선택]을 클릭하고 팝업 그림 선택 대화 상자에서 아이콘을 설정하고 기본 제공 아이콘 또는 로컬 그림을 선택할 수 있습니다.</p><ul><li>로컬 이미지를 선택할 때 .jpg/.jpeg/.png/gif/.ico.bmp/svg 형식의 이미지를 선택할 수 있습니다. .svg 형식의 그림을 선택할 때 그림의 색상을 설정할 수 있습니다. svg 그림의 오른쪽 상단 모서리를 클릭합니다.<img src="https://help.grapecity.com.cn/download/thumbnails/72352897/image2021-10-2_21-24-34.png?version=1&#x26;modificationDate=1648092538000&#x26;api=v2" alt=""></li></ul><ul><li>기본 제공 아이콘을 선택할 때 "셀 글꼴 색상 사용"을 선택하면 아이콘 색상이 텍스트 상자 셀의 텍스트 색상과 동일하도록 아이콘의 색상을 설정할 수도 있습니다.</li></ul></td></tr><tr><td>읽기전용</td><td>이 옵션을 선택하면 텍스트 상자를 편집할 수 없습니다.</td></tr><tr><td>셀 선택 시 전체 선택 상태로 표시</td><td>이 옵션을 선택하면 텍스트 상자에 포커스가 있으면 텍스트 내용이 모두 선택되고 텍스트 내용을 직접 편집할 수 있습니다.</td></tr></tbody></table>
