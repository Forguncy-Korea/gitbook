| [Reloaded](reloaded.md)                                                     | 이 이벤트는 리스트뷰 데이터를 다시 로드할 때 트리거됩니다.                                                                          |
| [SelectedRowsChanged](selectedrowschanged.md)                               | 이 이벤트는 사용자가 선택 열을 클릭할 때와 같이 리스트뷰의 선택 행이 변경될 때 트리거됩니다. 리스트뷰의 현재 행이 변경될 때만 트리거되는 SelectionChanged 이벤트와 다릅니다. |

| [ValueChanged](../cellevents/valuechanged.md)                               | <p>리스트뷰의 값이 변경될 때 이벤트가 트리거됩니다.<br></p>                                                                     |
