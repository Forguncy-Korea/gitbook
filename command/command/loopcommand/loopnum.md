# 지정된 횟수만큼 반복

루프 명령에서 하위 명령이 실행되는 지정된 횟수만큼 루프를 반복하도록 루프를 설정합니다.

![](<../../../.gitbook/assets/image (2291).png>)

## 지정된 횟수만큼 반복하기&#x20;

지정된 횟수만큼 반복하도록 선택한 후 고정 값, 수식을 입력하거나 셀을 선택할 수 있는 횟수를 설정해야 합니다. 횟수는 0보다 큰 값이어야 하며 값이 0보다 작거나 같으면 하위 명령이 실행되지 않습니다.

예를 들어 주문 목록 페이지의 루프 버튼에서 명령을 루프 명령으로 설정하고, 지정된 횟수만큼 반복하는 루프를 선택하고, 셀 T7의 값을 설정합니다.

![](<../../../.gitbook/assets/image (1809).png>)

다음 그림과 같이 루프의 하위 명령을 데이터 테이블 작업으로 설정하고 다음 그림과 같이 데이터 테이블 작업을 추가합니다.&#x20;

![](<../../../.gitbook/assets/image (1953).png>)

실행 후 주문 목록 페이지의 "루프" 버튼을 클릭하면 테이블 아래에 두 개의 새 레코드가 추가된 것을 볼 수 있습니다.

![](<../../../.gitbook/assets/image (1576).png>)
