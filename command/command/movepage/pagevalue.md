# 고급 설정 - 페이지 전송 값

리디렉션할 페이지가 내부 페이지, 즉 앱 내의 페이지인 경우 내부 페이지 간에 이동할 때 고급 설정을 수행할 수 있습니다.

고급 설정에서는 페이지가 이할 때 원본 셀의 값을 이동 후 페이지의 대상 셀에 전달하는 대상 페이지에 전달 값을 설정할 수 있습니다.

소스 페이지 및 대상 페이지, 예를 들어 페이지 1에서 버튼 클릭하고 페이지 2로 이동하면 페이지 1은 원본 페이지이고 페이지 1의 셀은 소스 셀입니다. 페이지 2는 대상 페이지이고 페이지 2의 셀은 대상 셀입니다.

![](<../../../.gitbook/assets/image (1795).png>)

## 대상 페이지로 값 전달&#x20;

페이지가 이동할 때 대상 페이지에 지정된 위치에 값을 전달할 수 있습니다. 새로 만들기 버튼 사용하여 소스 셀과 대상 셀을 추가하여 한 번에 여러 값을 대상 페이지에 전달할 수 있습니다.

소스 셀은 다음과 같은 입력 형식을 지원합니다.

* 고정 값입니다
* 원본 페이지의 셀입니다
* 소스 페이지의 테이블 템플릿 행의 셀: 페이지 이동 명령을 실행할 때 테이블이 현재 선택한 행에 해당하는 셀의 값을 전송합니다.
* 수식입니다

대상 셀은 다음과 같은 입력 형식을 지원합니다.

* 페이지에 지정된 셀: 대상 셀의 값을 전달 값으로 설정합니다.
* 테이블 템플릿 행의 셀: 대상 셀이 있는 열의 값이 전달 값과 동일한 행이 테이블에서 선택됩니다.

일반적으로 값 전달 기능을 사용하여 소스 페이지의 셀 값을 예제 1과 같이 대상 페이지에 전달합니다.

예제 1.&#x20;

주문 테이블의 버 명령을 페이지 이동으 설정하고 대상 페이지는 페이지 1이며 이동 때 주문 목록 페이지의 Z6셀 값을 이페이지  C4셀에 전달합니다.

![](<../../../.gitbook/assets/image (1903).png>)

실행 후 페이지 이 버튼을 클릭하면 주문 목록 페이지의 z6셀 값 "7"이 이동페이지 c4 셀로 전달됩니다.
