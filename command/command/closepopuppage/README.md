# 페이지 명령 - 팝업 페이지 닫기

팝업 및 닫기 팝업 명령은 쌍으로 나타나는 명령 쌍입니다. 팝업 페이지 닫기 명령을 사용하여 팝업 페이지를 닫고 팝업 페이지를 닫을 때 일부 설정을 수행할 수 있습니다.

## 명령 설정&#x20;

팝업 페이지를 닫을 때 다음과 같이 설정할 수 있습니다.&#x20;

| 설정                                                                                                                        | 설명                                                 |
| ------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------- |
| [입력 값을 상위페이지에 반환 ](toppage.md)                                                                                            | 팝업 페이지 명령에서 전달된 값을 원본 페이지로 다시 전달                   |
| 전[달받을 값 변경 발생 시 상위 페이지의 입력란에 반환 ](returninput.md)                                                                         | 팝업 페이지가 닫히면 이 옵션을 선택하면 부모 페이지가 바인딩된 데이터를 다시 로드     |
| [실행한 쿼리 결과를 상위 페이지에 전달 ](returnquery.md)                                                                                  | 팝업 페이지를 닫으면 팝업 페이지의 쿼리 조건이 데이터를 다시 로드하는 상위 페이지로 전달 |
| [고급 설정](https://www.grapecity.com.cn/solutions/huozige/help/docs/v10.0/command/closepopupwindowcommand/advanced-settings) | 고급 설정에서는 반환 결과와 반환 값을 설정할 수 있습니다.                  |
