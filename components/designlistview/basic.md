# 기본 속성

리스트뷰를 선택하면 리스트뷰의 이름을 지정하고 크기를 조정하는 리스트뷰의 기본 속성을 설정할 수 있습니다.

## 기본 속성&#x20;

리스트뷰를 선택한 후 리본의 리스트뷰 도구-디자인-속성 영역에서 리스트뷰 이름 및 리스트뷰 크기를 포함하여 리스트뷰의 속성을 설정할 수 있습니다.

![](<../../.gitbook/assets/image (1014).png>)

#### 리스트뷰 이름&#x20;

리스트뷰 이름 아래의 텍스트 상자에 리스트뷰 이름을 입력하거나 속성 설정 영역의 \[셀 설정] 탭에서 리스트뷰 이름을 설정할 수 있습니다.

![](<../../.gitbook/assets/image (1014).png>)

#### 리스트뷰 크기 조정

리스트뷰를 선택하고 \[크기 조정 시작]을 클릭하고 파란색 와이어프레임의 네 모서리를 드래그하여 리스트뷰의 크기와 위치를 수정하고 조정이 완료되면 \[크기 조정 종료]를 클릭합니다.

![](<../../.gitbook/assets/image (2194).png>)

리스트뷰의 마우스 오른쪽 버튼을 클릭하고 마우스 오른쪽 버튼을 클릭 메뉴에서 \[크기 조정 시작]을 선택하고 파란색 와이어프레임의 네 모서리를 드래그하여 리스트의 크기와 위치를 수정하고 조정이 완료되면 다른 영역을 클릭하여 조정을 종료할 수도 있습니다.

![](<../../.gitbook/assets/image (1672).png>)
