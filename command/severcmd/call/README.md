# 서버단 명령을 호출

서버단 명령을 만든 후 포건시 응용 프로그램 또는 다른 웹 응용 프로그램에서 브라우저에서 서버단 명령 호출을 요청할 수 있습니다.

서버 명령을 호출하는 방법은 다음과 같습니다.



![](https://help.grapecity.com.cn/download/thumbnails/72357439/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092606000\&api=v2)  [서버단 명령](../creadservercmd.md) 만들기를 참조하십시오.

![](https://help.grapecity.com.cn/download/thumbnails/72357439/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092606000\&api=v2) 완료되면 서버단 명령을 호출할 수 있으며 명령을 설정할 수 있는 모든 위치에서 호출할 수 있습니다.

예를 들어 페이지에서 셀 범위를 선택하고 버튼으로 설정합니다. 버튼의 명령을 편집하고 "서버단 명령 호출"으로 명령을 선택한 다음 서버단 명령 뒤에 있는 드롭다운을 클릭하여 드롭다운 목록에서 서버단 명령을 선택합니다.

* 파라미터 목록: 서버단 명령에 파라미터가 설정되어 있고 서버단 명령을 호출할 때 모든 파라미터가 자동으로 나열되며 파라미터 값을 고정 값으로 설정하거나 페이지의 셀을 선택할 수 있습니다.
* 반환 코드 : 페이지에서 셀을 선택할 수 있으며 서버단 명령을 성공적으로 호출하면 0을 반환합니다. 서버단 명령을 호출하지 못하면 명령 실행 실패에 대한 특정 오류 코드(예: 401, 500 등)가 반환됩니다.
* 반환 메시지 셀: 페이지에서 셀을 선택할 수 있으며 서버 쪽 명령이 성공적으로 호출되면 반환 정보가 없습니다. 서버 명령을 호출하지 못하면 명령 실행 실패에 대한 특정 오류 메시지가 반환됩니다.

![](<../../../.gitbook/assets/image (406).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72357439/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092606000\&api=v2) 설정이 완료되면 \[확인]을 클릭하여 대화 상자를 닫습니다. 페이지를 실행합니다.페이지에서 CSV 내보내기 버튼을 클릭하면 서비스 쪽에서 CSV 내보내기 작업이 수행되고 반환 코드 0이 페이지에 지정된 셀로 반환되어 명령 실행이 성공합니다.

CSV 내보내기 버튼 클릭하면 사용자가 호출할 수 있는 권한이 없는 경우 반환 코드 401과 반환 정보가 페이지에 지정된 셀에 표시되어 사용자에게 서버단 명령을 실행할 수 있는 권한이 없음을 나타냅니다.
