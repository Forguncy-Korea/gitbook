# IMergedColumnInfo 인터페이스

테이블의 열 정보를 나타냅니다. getMergedColumnInfos 메서드 에 사용합니다.



| 속성                  | 설명                      |
| ------------------- | ----------------------- |
| ColumnName          | 열 이름                    |
| ColumnType          | 열 유형                    |
| DesignerColumnCount | 디자이너에 있는 열의 병합된 열 수입니다. |
| DesignerColumnIndex | 디자이너의 첫 번째 열 인덱스입니다.    |
