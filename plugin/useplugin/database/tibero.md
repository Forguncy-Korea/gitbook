# Tibero 데이터베이스

포건시는 데이터베이스 플러그인(Tibero 등)은 플러그인 형태로 제품에 내장되어 있어 수동으로 설치할 필요가 없습니다.&#x20;

> Tibero  데이터베이스는 읽기만 가능합니다. Tibero 측 사정으로 OLEDB의 버전 업그레이드 지원이 종료되었습니다.&#x20;
