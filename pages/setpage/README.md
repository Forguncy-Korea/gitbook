# 페이지 설정

포건시를 사용하여 정보 시스템의 세 가지 중요한 구성 요소인 웹 페이지, 데이터베이스 및 명령을 개발합니다.

웹 페이지는 웹 페이지의 인터페이스입니다. 웹 정보 시스템은 각 페이지에 관련 데이터 및 비즈니스 프로세스가 포함된 페이지 집합으로 구성되므로 먼저 웹 페이지가 있어야 합니다. 포건시의 한 페이지에는 웹 페이지 페이지가 있습니다.

## 페이지

포건시에는 일반 페이지, 휴대폰 페이지, 마스터 페이지 및 기본 제공 페이지가 포함됩니다.

**일반 페이지**&#x20;

일반 페이지는 페이지 페이지입니다. 일반 페이지를 설정하고 관리할 수 있습니다.

페이지 관리: 연결된 페이지 집합을 폴더에 배치하는 폴더를 만들 수 있습니다.

**모바일 페이지**

모바일 페이지는 모바일 브라우저에서 액세스하는 데 사용되는 페이지입니다.

**마스터 페이지**

마스터 페이지는 여러 일반 페이지(예: 마스터 페이지에서 탐색 모음 디자인)와 함께 사용할 수 있는 공유 섹션을 디자인하는 데 사용됩니다.

마스터 페이지는 일반 셀 범위와 페이지 자리 매김의 두 부분으로 나뉩니다. 일반 셀 범위에서 탐색 모음과 같은 공유 섹션을 디자인할 수 있으며 페이지 자리 표시자는 일반 페이지 표시를 전환하는 데 사용됩니다.

**내장 페이지**

포건시에서 모든 템플릿 또는 새 빈 프로젝트를 만들면 기본 제공 페이지와 기본 제공 휴대폰 페이지를 비롯한 일부 기본 페이지가 있음을 알 수 있습니다.

내장 된 페이지는 특별한 시간에 표시되는 포건시 특정 기능에 있습니다. 시스템이 401의 경우, 401 페이지가 자동으로 이동, 사용자가 로그인해야 하는 경우 로그인 페이지로 자동으로 이동합니다.

기본 제공 페이지의 인터페이스와 모양을 수정할 수 있지만 로그인 페이지의 사용자 이름, 암호 및 로그인 단추와 같은 특수 기능이 있는 텍스트 상자, 버튼 또는 하이퍼링크는 삭제할 수 없습니다.
