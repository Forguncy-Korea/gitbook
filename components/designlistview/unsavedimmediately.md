# 즉시 저장 해제



포건시에서 데이터는 기본적으로 밀접하게 바인딩된 방식으로 제출되며, 이는 리스트뷰의 데이터가 수정되는 즉시 데이터베이스에 커밋됩니다.

리스트뷰 설정에서 데이터 제출 모드를  즉시 저장 해제로 변경할 수 있습니다.

## 즉시 저장 해제&#x20;

데이터 제출 방법을 "즉시 저장 해제"로 변경하면 새 레코드 추가, 기존 레코드 수정 또는 삭제를 포함하여 리스트뷰의 모든 작업이 즉시 데이터베이스에 업데이트 되지 않습니다.

* 페이지 테이블 작업이 완료되면 데이터 명령 테이블 작업 명령을 사용하여 데이터베이스에 대한 모든 업데이트를 제출할 수 있습니다.
* 페이지에서 마스터-종속 테이블을 사용하는 경우 이 옵션을 선택하면 테이블에서 수행한 작업이 즉시 데이터베이스로 업데이트되지 않습니다. 데이터 명령-데이터 테이블 작업 명령을 사용하여 기본 테이블 데이터를 데이터베이스로 업데이트하는 경우에만 에서 마스터 테이블 데이터와 함께 데이터베이스로 업데이트됩니다.

리스트뷰 선택하고 마우스 오른쪽 버튼 클릭하고 마우스 오른쪽 버튼 클릭 메뉴에서 \[리스트뷰 옵션 설정]을 선택하고  \[데이터] 탭에서 데이터 제출에서 \[즉시 저장 해제] 선택합니다.
