# 사용자 정의 컨트롤 컨테이너

사용자페정의이컨트롤지컨에티너는 컨테이너 셀과 유사하게 구성 요소 페이지를 표시하는 데 사용됩니다.

페이지에서 영역을 선택하고 해당 셀 유형을 사용자 정의  컨트롤 컨테이너 셀로 설정하십시오.이 영역은 구성 요소 페이지를 표시하는 데 사용됩니다.구성 요소 페이지의 크기에 따라 영역의 크기를 적절하게 조정하십시오.<br>

## 설정&#x20;

속성 설정 영역에서 "셀 설정" 탭을 선택합니다. 설정 지침은 아래 표를 참조하십시오.<br>

| 설정             | 설명                                                                                                                                                                                                                                                                                         |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 사용자 정의 컨트롤 이름  | 컴포넌트 컨테이너에 표시되는 컴포넌트 페이지를 설정합니다.                                                                                                                                                                                                                                                           |
| 이벤트 명령 편집      | <p>구성요소에 대한 이벤트를 생성한 후 이벤트의 명령을 편집할 수 있습니다.</p><p>명령을 설정하면 하이퍼링크 텍스트가 굵게 표시됩니다.</p>                                                                                                                                                                                                        |
| 속성             | 구성 요소의 속성을 생성한 후 속성 값을 설정할 수 있습니다.                                                                                                                                                                                                                                                         |
| 하위 페이지 오버플로 모드 | <p>컴포넌트 페이지를 선택한 후 컴포넌트 페이지의 오버플로 모드를 설정할 수 있으며, 이 옵션을 사용하여 컴포넌트 페이지 표시 방법을 설정할 수 있습니다.</p><p>세 가지 모드가 있습니다:</p><ul><li>오버플로: 구성 요소 페이지가 완전히 표시될 수 있을 때까지 구성 요소 컨테이너가 확장됩니다. </li><li>스크롤: 구성 요소 컨테이너에 스크롤 막대가 나타납니다.</li><li>잘라내기: 구성 요소 컨테이너를 초과하는 구성 요소 페이지 부분이 잘려 표시되지 않습니다.</li></ul> |
| 노트             | 구성 요소 컨테이너 셀에 대한 설명 자세한 내용은 [기본 셀 작업](https://www.grapecity.com.cn/solutions/huozige/help/docs/v10.0/celltypes/celltypeoperations) 에 대한 "새 설명"을 참조하세요.                                                                                                                                     |
| 사용 가능한 작업      | <p>구성 요소 컨테이너에서 사용할 수 있는 작업은 속성 값 설정, 속성 값 가져오기, 구성 요소 설정입니다. 사용 가능한 작업을 페이지로 끌어서 셀 작업 명령을 생성합니다.</p><p><img src="../../.gitbook/assets/image (2448).png" alt="" data-size="original"></p>                                                                                                 |
