# Excel명령 - Excel 을 리스트뷰로 가져오기

Excel 데이터를 리스트뷰로 가져오기 명령을 사용하여 Excel의 데이터를 테이블로 직접 가져올 수 있습니다.<br>

<figure><img src="../../.gitbook/assets/image (2514).png" alt=""><figcaption></figcaption></figure>

## 명령 설정&#x20;

Excel 데이터를 리스트뷰로 가져오기 명령에서 데이터를 가져올 리스트뷰를 선택하면 리스트뷰의 템플릿 행의 셀과 해당 머리글이 템플릿 열 및 템플릿 열 표시 이름으로 자동으로 채워집니다.

템플릿 열 셀 및 템플릿 열의 표시 이름을 사용자 지정하려면 \[템플릿 열 추가]를 클릭합니다.

Excel의 열 이름은 가져온 Excel의 열 이름과 정확히 동일해야 합니다. 그렇지 않으면 오류 메시지가 표시됩니다. Excel의 열 이름은 개행 입력을 지원합니다.</li></ul></td></tr></tbody></table>

여기서 Excel의 열 이름인 헤더 행은 첫 번째 행에 있어야 합니다. Excel의 열 이름은 다음 그림과 같이 페이지 테이블의 열 이름과 정확히 일치해야 합니다.

## Excel 데이터를 테이블로 가져오기&#x20;

일반적으로 Excel 데이터를 리스트뷰로 가져오는 명령은 파일 다운로드 명령과 함께 사용됩니다. 사용자가 페이지에서 템플릿 파일을 다운로드한 후 데이터를 채우고 페이지에 업로드합니다.

다음은 두 명령을 함께 사용하는 절차를 보여 주는 예제입니다.

![](https://help.grapecity.com.cn/download/thumbnails/72354196/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092557000\&api=v2) 주문 목록 페이지에서 버튼을 추가하고 명령을 파일 다운로드로 설정하고 다운로드 가능한 파일인 주문 템플릿 .xlsx 업로드합니다. 자세한 내용은 [Excel 명령 - 템플릿 파일 다운로드](excel.md)를 참조하십시오.

![](<../../.gitbook/assets/image (1087).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72354196/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092557000\&api=v2) 주문 목록 페이지에 추가합니다.\[Excel 가져오기] 버튼을 사용하여 \[Excel을 리스트뷰로 가져오기]로 명령을 설정하고 가져온 테이블 \[리스트뷰 1] 및 가져오기 모드 \[추가]를 선택합니다. 리스트뷰 1을 선택하면 템플릿 열과 템플릿 열  이름이 자동으로 채워집니다.

![](<../../.gitbook/assets/image (1124).png>)

{% hint style="info" %}
* 테이블의 콤보 상자의 경우 데이터베이스에서 프로젝트를 빌드하도록 선택하고 값 필드를 선택하고 표시 필드를 다르게 선택하면 포건시가 표시 필드별로 인식됩니다.
* 가져온 Excel 파일에서 셀 A와 셀 B가 병합되면 테이블로 가져온 셀 A와 셀 B의 값이 동일합니다.
* 테이블을 편집할 수 없는 경우에도 Excel 데이터를 테이블로 가져올 수 있습니다.
* Excel 가져오기 명령은 명령 목록의 첫 번째 명령이어야 하며 조건부 및 순환 명령에서 사용할 수 없습니다.
* 가져오기 .xlsx 형식 파일만 지원되며 해당 열의 데이터 형식은 일관되어야 합니&#xB2E4;**.**
{% endhint %}

## 가져오기 모드&#x20;

Excel 데이터를 테이블로 가져올 때 세 가지 가져오기 모드가 있습니다.

* 추가: Excel의 데이터를 테이블의 원래 데이터를 기반으로 추가합니다.
* 합치기 : Excel과 테이블의 데이터를 병합합니다.
* 교체 : Excel 기반 데이터로 테이블의 데이터를 덮어씁니다.

**추가**

Excel의 데이터를 테이블의 원래 데이터를 기반으로 추가합니다.&#x20;

**합치기**

병합 모드를 선택할 때 하나 이상의 열을 기본 키로 지정해야 하며 Excel 데이터를 가져올 때 활자 격자는 기본 키 열 데이터를 쿼리 정보로 사용하여 Excel 및 테이블의 데이터를 비교합니다.

Excel과 테이블에 모두 있는 데이터는 Excel의 데이터로 업데이트되고 나머지는 테이블에 병합된 상태로 유지됩니다.

선택 모드는 병합, 기본 키로 주문 번호 설정, Excel에서 데이터를 가져온 후 다음과 같이 표시됩니다.

![](<../../.gitbook/assets/image (2141).png>)



**교체**

대체 모드를 선택할 때 하나 이상의 열을 기본 키로 지정해야 하며 Excel 데이터를 가져올 때 포간시는 기본 키 열 데이터를 쿼리 정보로 사용하여 Excel 및 테이블의 데이터를 비교합니다.<br>

대체 모드는 Excel의 데이터를 기반으로 테이블의 데이터를 덮어씁니다.

* Excel에 없지만 테이블에있는 데이터를 삭제합니다.
* Excel에 있지만 테이블에 없는 데이터를 추가합니다.
