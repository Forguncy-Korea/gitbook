# Excel 가져오기 및 내보내기

사용자의 계정 정보는 Excel 파일로 내보내기를 지원하며 사용자 계정 정보에 대한 Excel 템플릿 파일에 사용자 정보를 입력하고 포건시로 가져올 수 있습니다.

이 섹션에서는 사용자 계정 정보 Excel의 가져오기 및 내보내기에 대해 설명하겠습니다.&#x20;



## Excel 파일 내보내기&#x20;

사용자 계정 정보를 사용하면 사용자 역할, 조직 구조 등을 포함하여 Excel 파일을 내보낼 수 있습니다.

서버 관리자의 \[사용자 설정] 영역이 내장되어 있는 경우 \[가져오기 및 내보내기]를 클릭하고 \[내보내기]를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2422).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
내보낸 Excel 파일의 기본 이름은 사용자 이름, 사용자 유형, 전체 이름, 전자 메일 주소, 암호, 역할, 소유 조직 정보 및 기타 사용자 지정 속성 정보가 포함된 "Forguncy 사용자 .xlsx"입니다.
{% endhint %}

## Excel 파일 가져오기&#x20;

사용자 계정 정보를 Excel로 내보낸 후 Excel 파일을 가져올 수도 있습니다. Excel 템플릿 파일에 사용자 정보를 입력한 후 가져올 수도 있으며, 이 정보를 포건시로 가져올 수도 있습니다. Excel 템플릿 파일은 Excel 파일 내보내기에서 사용할 수 있습니다.

서버 관리자의 \[사용자-설정] 영역에서 \[가져오기 및 내보내기]를 클릭하고  \[가져오기]를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2423).png" alt=""><figcaption></figcaption></figure>

Excel 파일을 가져올 때 다음 두 가지 모드를 선택할 수 있습니다.

* 가져오기 병합: 병합 모드로 Excel을 가져옵니다. 원래 사용자 및 사용자의 역할, 사용자 정의 속성 및 사용자가 속한 조직을 유지합니다. 원본 데이터베이스에서는 역할 또는 조직을 삭제하지 않습니다. 조직 수준 및 역할의 인증에는 영향을 미치지 않습니다.
* 가져오기 재정의: 재정의 모드로 Excel을 가져옵니다. Excel에 있는 사용자의 정보를 유지하고 Excel에 없는 원래 사용자를 삭제합니다(관리자 제외). 원본 데이터베이스에서는 역할 또는 조직을 삭제하지 않으며 병합 모드와 같이 조직 수준 및 역할의 인증에는 영향을 미치지 않습니다.

{% hint style="info" %}
- **Excel 파일의 사용자 이름이 비어 있으면 가져오지 않습니다.**
- **전체 이름은  forguncy 인증 사용자의 전체 이름을 재정의하지만 Windows 도메인 사용자의 전체 이름에는 영향을 주지 않습니다.**
- **Excel 파일의 암호가 비어 있는 경우:**
  * **사용자가 이미 있는 경우 사용자의 원래 암호를 유지합니다.**
  * **사용자가 새 사용자인 경우 암호가 기본 암호로 설정된 123456**
  * **암호는 도메인 사용자에게 영향을 주지 않습니다.**
- **도메인 사용자 계정을 사용하여 사용자 관리 플랫폼에 로그인할 수 없습니다.**
- **추가된 도메인 사용자가 도메인에 없으면 빨간색으로 표시됩니다. 제대로 사용하기 전에 도메인에 추가할 수 있습니다.**
- **가져온 사용자 중 두 명의 사용자 이름과 사용자 유형이 동일한 경우 전체 Excel 파일을 가져오지 않습니다.**
- **사용자가 동일한 조직 구조 아래에 두 개 이상의 역할이 있는 경우 마지막 비어 있지 않은 역할이 사용자의 조직 역할로 처리되고 나머지 새 역할이 역할 목록에 추가됩니다. 동일한 조직 경로에서 사용자를 리더로 나타내는 문자열과 사용자가 이끌지 않은 문자열이 모두 있는 경우 마지막 문자열을 사용하여 사용자가 리더인지 여부를 결정합니다.**
{% endhint %}

## 포건시 사용자 파일&#x20;



내보낸 Excel 파일 이름은 Forguncy 사용자 .xlsx 입니다.&#x20;

Forguncy 사용자 .xlsx 사용하여 사용자 정보를 입력할 때 주의해야 할 몇 가지 사항은 다음과 같습니다.

* 빨간색 필드는 필수이고 다른 필드는 선택 사항입니다. "사용자 이름"과 "사용자 유형"은 사용자의 고유 ID를 구성합니다.
* 암호: 암호를 비워 두면 기본값은 "123456"입니다.
* 역할: 세미콜론으로 구분된 여러 역할;
* 조직: 상위 및 하위 레벨은 최상위 레벨부터 시작하여 "/"로 구분됩니다. 조직 역할이 있는 경우 조직의 마지막 수준 뒤에 꺾쇠 괄호로 묶인 역할 이름을 추가해야 합니다. 사용자가 조직의 리더인 경우 끝에 "#leader"를 추가합니다. 예: "회사/R\&D 부서/R\&D 그룹 <그룹 리더>, 특정 회사/테스트 부서#리더". 사용자가 여러 조직에 속해 있으면 세미콜론으로 구분하고, 조직이 없으면 자동으로 조직이 생성되어 사용자가 조직에 추가되고, 조직이 비어 있으면 사용자가 어느 조직에도 속하지 않습니다.
* H 열(H 열 포함) 이후의 열은 사용자 지정 속성입니다. 열 이름 앞에 "#"을 추가하면 열을 사용자 지정 속성으로 가져올 수 있습니다. 열 이름 앞에 "#"을 추가하지 않으면 열을 사용자 지정 속성으로 가져올 수 있습니다.
* 사용자 지정 속성 이외의 열 이름은 Excel 테이블에서 추가, 삭제 또는 수정할 수 없습니다.
* 모든 사용자 정보는 Users 양식에 입력해야 하며 초과분은 가져올 수 없습니다. 사용자 또는 속성을 추가할 때 테이블 크기를 적절하게 조정합니다.
