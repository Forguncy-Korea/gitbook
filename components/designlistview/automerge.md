# 자동 병합

실제로 리스트뷰 내의 동일한 셀을 병합하여 데이터를 보다 명확하게 보이게 해야 하는 경우가 많습니다.

포건시의 리스트뷰는 자동 병합을 지원하며 자동 병합이 켜져 있으면 동일한 값을 가진 다른 행의 인접 셀이 병합됩니다.

전체 리스트뷰에 대해 자동 병합을 설정하거나 열에 자동 병합을 설정할 수 있습니다.

* 리스트뷰를 선택하고 리본의 리스트뷰 도구-디자인-열 설정 영역에서 자동 병합을 선택하면 자동으로 병합된 개체가 전체 테이블입니다.
* 리스트의 첫 번째 열에 인접하지 않고 동일한 값을 셀을 병합할 수 있는 경우 다음 열에는 인접하고 동일한 값이 있는지 여부에 관계없이 셀이 병합되지 않습니다.
* 리스트뷰의 첫 번째 열에 인접하고 동일한 값이 있으면 셀이 병합되고 다음 열에 병합을 위한 조건이 충족된 셀이 있는지 차례로 확인합니다. 열이 병합 조건을 충족하지 않는 경우 다음 열은 더 이상 자동으로 병합되지 않습니다.

예를 들어 리스트뷰를 선택한 후 자동 병합을 선택하면 첫 번째 열에 인접한 행과 동일한 값의 셀이 병합된 것을 볼 수 있습니다.
