# 상태정보

리스트뷰 명령에서 상태정보 선택 시 상태 정보 항목에는 리스트뷰 상태 저장과 리스트뷰 상태 로드 두가지 동작 종류가 있습니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (2508).png" alt=""><figcaption></figcaption></figure>



### 리스트뷰 상태 저장&#x20;

이 작업은 리스트뷰의 열, 열 너비 및 열 순서를 포함하여 리스트뷰의 상태를 저장합니다. 예를 들어, 열 옵션 명령을 통해 리스트뷰에 표시된 열을 변경하는 경우 이 명령을 사용하여 리스트뷰 상태를 저장할 수 있습니다.

열 헤더의 필터링 및 정렬과 쿼리 대화 상자의 쿼리 조건을 저장할 수도 있습니다.

이름은 비어 있거나 사용자 정의될 수 있습니다. 리스트뷰는 기본적으로 빈 테이블 상태로 로드됩니다.

테이블에 저장된 테이블 상태가 표시되도록 하려면 리스트뷰상태 로드 명령을 사용합니다.

<figure><img src="../../../.gitbook/assets/image (2509).png" alt=""><figcaption></figcaption></figure>



### 리스트뷰 상태로드&#x20;

저장된 양식 상태는 이름을 지정하여 로드할 수 있습니다. 이름은 비어 있거나 사용자 정의될 수 있습니다.
