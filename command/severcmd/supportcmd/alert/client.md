# 클라이언트 명령

클라이언트 명령에서 아래 그림과 같이 새로운 서버 알림 명령 세트가 추가되었습니다.

![](<../../../../.gitbook/assets/image (1070).png>)

### 서버 알림 명령 구독&#x20;

선택 명령은 서버 알림 명령을 구독하는 것으로, 서버 알림의 이름을 선택해야 하며 짧은 시간 내에 동일한 알림을 병합할지 여부도 설정할 수 있습니다.

<figure><img src="../../../../.gitbook/assets/image (618).png" alt=""><figcaption></figcaption></figure>

서버 알림 명령어를 구독한 후 서버 알림을 받은 후 명령어를 설정할 수 있습니다. 예를 들어 다음 두 가지 명령을 추가하십시오. 즉, 서버에서 알림을 받은 후 페이지에 메시지 상자가 팝업되고 메시지 상자를 닫은 후 페이지가 이동합니다.

<figure><img src="../../../../.gitbook/assets/image (1067).png" alt=""><figcaption></figcaption></figure>

### 구독 취소 알림 명령&#x20;

선택 명령은 구독 취소 알림 명령으로,    서버 알림의 이름을 선택해야 합니다.
