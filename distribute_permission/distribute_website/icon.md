# 웹사이트 아이콘 설정

브라우저에서 웹 사이트를 열면 탭의 시작 부분에 있는 아이콘이 웹 사이트의 아이콘입니다. 포건시를 사용하여 만든 사이트의 기본 아이콘은 설정 사이트의 아이콘을 사용할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (1540).png" alt=""><figcaption></figcaption></figure>

\[파일-> 옵션-> 응용 프로그램 설정]을 선택하고 \[주소창에 표시되는 파비콘] 영역에서 기본 아이콘 뒤에 있는 아이콘을 클릭하여 변경할 아이콘을 선택합니다.
