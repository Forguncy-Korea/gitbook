# SQL 실행 명령

포건시는 서버 측에서 SQL 명령을 실행하고, SQL 문을 통해 데이터베이스를 조작하고, 저장 프로시저 작성 및 호출, 임시 테이블 만들기, 테이블 인덱스 만들기, 여러 쿼리 만들기 등과 같은 복잡한 기능을 수행할 수 있도록 지원합니다.

## 명령 실행&#x20;

다음은 SQL Server를 호출하는 저장 프로시저를 예로 들어 서버 쪽 명령의 저장 프로시저를 사용하여 명령을 호출하는 방법을 보여 줍니다.

![](https://help.grapecity.com.cn/download/thumbnails/72358449/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1648092619000\&api=v2)프로젝트 탐색기에서 서버단 명령 탭을 마우스 오른쪽 버튼을 클릭하고 서버단 명령 만들기를 선택하고 서버단 명령 편집 대화 상자를 표시하고 서버 명령의 일반 설정을 편집합니다. 서버 쪽 명령의 이름을 "SQL 실행 명령" 으로 설정합니다.

![](https://help.grapecity.com.cn/download/thumbnails/72358449/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1648092619000\&api=v2) 서버단 명령을 편집하는 명령입니다. \[명령 편집] 하이퍼링크를 클릭하고 서버 명령 편집 대화 상자를 표시하고 명령을 \[SQL 실행 명령]으로 선택합니다.

* 연결 문자열: 연결 문자열을 선택하고 연결 문자열이 없는 경우 연결 문자열 관리를 클릭하여 직접 연결할 수 있습니다. 드롭다운 버튼을 클릭하여 연결 문자열을 선택합니다. 내부 라이브러리를 선택할 수도 있습니다.
* 매개 변수 목록: 매개 변수 목록에서 쿼리 매개 변수를 설정합니다.
* SQL 문: SQL 문을 입력합니다. SQL 문 편집을 클릭하면 SQL 문 편집 대화 상자가 열리고 이 대화 상자에서 키워드와 일반적으로 사용되는 SQL 문을 선택하여 사용할 수 있습니다.
* 변수에 값 반: 반환 값을 반환할 매개 변수를 설정합니다. SQL 명령을 실행하면 반환 값이 지정된 매개 변수로 반환됩니다.
* 고급 설정: 고급 설정에서 실행 시간 제한 및 캐시 만료 시간을 설정할 수 있습니다.

![](<../../../.gitbook/assets/image (1160).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72358449/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1648092619000\&api=v2)  설정완료되면 이 서버단 명령을 호출할 수 있습니다.

예를 들어 페이지에서 셀 범위를 선택하고 버튼을 설정합니다. 버튼의 명령을 편집하고 "서버단 명령 호출"으로 명령을 선택한 다음 서버 쪽 명령 뒤에 있는 드롭다운을 클릭하고 드롭다운 목록에서 "SQL 실행 명령"을 선택합니다.

![](<../../../.gitbook/assets/image (714).png>)

![](https://help.grapecity.com.cn/download/thumbnails/72358449/%E6%AD%A5%E9%AA%A44.png?version=1\&modificationDate=1648092619000\&api=v2) 설정이 완료되면 \[확인]을 클릭하여 대화 상자를 닫습니다. 페이지를 실행합니다.주문 삭제 버튼을 클릭하면 서버단 명령이 호출되고 SQL 작업이 수행되어 주문 번호 A001의 주문이 삭제됩니다.
