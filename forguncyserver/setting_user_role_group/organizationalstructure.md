# 조직 구조 만들기&#x20;

## 조직구조

조직 구조는 서로 다른 조직 노드 간의 관계를 정의하고 각 조직 노드는 조직 수준을 정의할 수 있습니다.

조직 구조는 조직의 계층적 관계를 나타내는 트리 구조(상하 관계를 나타낼 수 있음) 트리의 상위 조직 노드에 있는 사용자는 하위 사용자의 조직 상사를 나타냅니다. 트리의 조직 ​​노드 트리에 있는 각 조직 노드의 구체적인 의미는 조직 수준에 따라 결정됩니다. 동일한 조직 노드의 사용자는 피어입니다.

조직 구조가 설정되거나 수정되면 즉시 적용됩니다.

## 조직 수준&#x20;

조직 수준은 조직 구조에서 조직 노드의 수준 식별입니다.

회사나 조직마다 조직 레벨이 다르므로 트리 형태의 조직 구조를 구성하려면 조직 노드에 대한 조직 레벨을 설정하면 됩니다.

#### 조직의 수준을 높이다 <a href="#id-chuang-jian-zu-zhi-jie-gou-zeng-jia-zu-zhi-ji-bie" id="id-chuang-jian-zu-zhi-jie-gou-zeng-jia-zu-zhi-ji-bie"></a>

관리 콘솔의 "내장 사용자-사용자 관리" 영역에서 "조직 수준"을 선택하고 "조직 수준 추가"를 클릭하고 팝업 대화 상자에 조직 수준의 이름을 입력하여 추가합니다. 새로운 조직 수준.

## 조직 구성원&#x20;

조직 구조 및 조직 수준을 설정한 후 조직 구성원을 설정해야 합니다.

#### 구성원 추가 <a href="#id-chuang-jian-zu-zhi-jie-gou-tian-jia-cheng-yuan" id="id-chuang-jian-zu-zhi-jie-gou-tian-jia-cheng-yuan"></a>

조직 노드에 조직 구성원을 추가합니다.

**단계**

![](https://help.grapecity.com.cn/download/thumbnails/80944269/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1673923140000\&api=v2)  조직 노드를 선택하고 오른쪽에서 사용자 추가를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1403).png" alt=""><figcaption></figcaption></figure>

![](https://help.grapecity.com.cn/download/thumbnails/80944269/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1673923140000\&api=v2)  팝업된 "멤버추가" 대화상자에서 사용자 인증 모드를 선택하고 검색창에 검색어를 입력한 후 클릭![](https://help.grapecity.com.cn/download/thumbnails/80944269/image2023-2-24_11-25-6.png?version=1\&modificationDate=1677209106000\&api=v2)또는 Enter 키를 눌러 검색하거나 사용자를 직접 선택할 수 있습니다.

사용자를 성공적으로 추가하려면 확인을 클릭하십시오. 여러 번 검색하고 마지막 사용자의 확인란을 선택한 다음 마지막으로 "확인"을 클릭하면 선택한 모든 사용자를 한 번에 추가할 수 있으므로 사용자를 추가하는 데 편리합니다.

<figure><img src="../../.gitbook/assets/image (1977).png" alt=""><figcaption></figcaption></figure>

![](https://help.grapecity.com.cn/download/thumbnails/80944269/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1673923139000\&api=v2)   추가하면 해당 조직 노드 아래에 사용자가 표시됩니다.

<figure><img src="../../.gitbook/assets/image (588).png" alt=""><figcaption></figcaption></figure>

#### 감독자로 설정 <a href="#id-chuang-jian-zu-zhi-jie-gou-she-zhi-wei-zhu-guan" id="id-chuang-jian-zu-zhi-jie-gou-she-zhi-wei-zhu-guan"></a>

"선택한 사용자를 감독자로 설정"을 체크하면 추가 완료 후 해당 사용자는 조직 노드의 감독자가 되며, 조직 노드 아래의 다른 사용자보다 권한이 높지만 여전히 상위 사용자보다 낮습니다.&#x20;

조직 구성원 행에서 "리더" 작업 열 아래의 확인란을 선택하여 사용자가 조직 노드의 감독자임을 나타냅니다.
