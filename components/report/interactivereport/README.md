# 대화형 보고서

기존의 보고서는 주로 데이터 표시, 인쇄 및 내보내기 기능을 중심으로 비교적 정적 데이터 표현, 보조 의사 결정 서비스입니다. 최신 비즈니스 인텔리전스는 데이터를 지능화하고 분석해야 하며, 기존의 보고서는 비즈니스 인텔리전스의 요구를 충족시키지 못하는 경우가 많습니다.

대화형 보고서는 지능형 비즈니스 분석을 위한 보고 솔루션이며, 정적 보고서를 가능한 한 동적으로, 즉 보고서 데이터를 동적으로 만들고 보고서 형식을 동적으로 만들어 보고서의 실제 사용 가치를 향상시킵니다. 사용자의 분석 각도 및 데이터 선택에 따라 보고서 프레젠테이션이 다릅니다.

대화형 보고서는 사용자가 필요에 따라 데이터 렌더링을 동적으로 정의할 수 있는 대화형 기능을 제공하고 링크를 클릭하여 다른 보고서 또는 개체에 액세스할 수 있습니다.

즉, 대화형 보고서는 정적 보고서에 사용자 실행 기능을 추가하여 보고서를 대화형으로 만듭니다.
