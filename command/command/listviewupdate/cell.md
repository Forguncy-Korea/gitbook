# 셀작동&#x20;

리스트뷰  명령  작동에서 동작 타입을 선택하면 동작 셀 카테고리 아래에 4개의 동작 타입이 있습니다

<figure><img src="../../../.gitbook/assets/image (2001).png" alt=""><figcaption></figcaption></figure>



## 값 가져오기&#x20;

특정 셀의 값을 가져옵니다.

취득할 셀 값의 행 번호, 열 번호/열 이름을 설정해야 하며, 설정 후 셀의 값을 취득하여 변수에 저장하여 다음 작업에서 사용할 수 있습니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (1668).png" alt=""><figcaption></figcaption></figure>

## 텍스트 가져오기&#x20;

특정 셀의 텍스트를 가져옵니다.

취득할 셀 텍스트의 행 번호, 열 번호/열 이름을 설정해야 하며, 설정 후 셀의 텍스트를 취득하고 그 값을 변수에 저장하여 다음 작업에서 사용할 수 있습니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (445).png" alt=""><figcaption></figcaption></figure>

## 값 설정&#x20;

특정 셀의 값을 설정합니다.셀의 행 번호, 열 번호/열 이름 및 값을 설정해야 합니다.

<figure><img src="../../../.gitbook/assets/image (1719).png" alt=""><figcaption></figcaption></figure>

## 텍스트 설정&#x20;

특정 셀의 텍스트를 설정합니다. 셀의 행 번호, 열 번호/열 이름 및 텍스트를 설정해야 합니다.
