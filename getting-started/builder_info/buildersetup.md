# 빌더(Builder) 설치

Forguncy Builder는 Excel 화면을 기반으로 원하는 웹 페이지를 디자인하고 다양한 기능을 구현할 수 있는 노코드(No-Code) 기반의 웹 개발 도구입니다.



아래와 같은 방법으로 Forguncy Builder를 설치할 수 있습니다.

{% hint style="info" %}
시스템 요구사항

**포건시 빌더** 설치 전에 반드시 [**시스템 요구사항**](../systemrequirement.md)을 확인해주세요.
{% endhint %}



**1. 먼저 공식 홈페이지에서 포건시를** [**다운로드**](https://www.mescius.co.kr/forguncy/download) **받습니다.**



**2. 다운로드가 완료되면, 다운로드 받은 Forguncy\*-KR-20xx.xxxx.x.exe를 더블 클릭합니다.**          \
&#x20;**(본 매뉴얼은 해당 10 버전을 기준으로 설명합니다.)**&#x20;



**3. 아래와 같은 Forguncy Builder 설치 화면이 나타납니다.**

(1) “사용자 라이선스 동의서를 확인하였으며 이에 동의함”에 동의하셔야 설치가 진행됩니다.

(2) 대상폴더(설치경로)는 기본으로 두셔도 되고, 사용자가 필요한 위치로 옮기셔도 됩니다.

(3) EULA(사용자 라이선스 동의서)에 동의하시면 “설치” 버튼이 활성화됩니다. 클릭하셔서 설치를 진행해 주세요.

<figure><img src="../../.gitbook/assets/image (2495).png" alt=""><figcaption></figcaption></figure>

**4. 설치가 완료된 후, Forguncy Builder를 실행하실 수 있습니다.**

<figure><img src="../../.gitbook/assets/image (2497).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**문의하기**

설치 중 오류, 혹은 설치 후 Foruncy Builder 실행 관련하여 문의사항이 있으신 경우,\
메시어스**개발자 포럼으로 문의**를 남겨주세요.\
\
<mark style="color:orange;">**\* 오류가 발생한 경우에 대한 상황을 스크린샷 찍어주시거나, 명확히 설명해 주시면 빠른 문제 해결에 도움이 됩니다**</mark>
{% endhint %}
