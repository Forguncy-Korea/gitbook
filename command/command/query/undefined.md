# 고급 검색

쿼리 명령에서 여러 쿼리 조건을 추가하거나 여러 쿼리 조건을 연결하거나 연결할 수 있습니다.

또한 괄호를 사용하여 최대 4단계까지 조건을 묶을 수 있습니다. 괄호를 사용하면 "((A 및 B) 또는 (C 및 D)"와 같은 복잡한 표현을 설정할 수 있습니다.

쿼리 명령을 사용하여 여러 쿼리 조건을 추가하고 및/또는 대괄호를 사용하여 여러 쿼리 조건을 연결하여 고급 쿼리를 수행합니다.
