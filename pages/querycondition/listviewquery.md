# 리스트뷰 쿼리

리스트뷰에 대한 쿼리 조건을 설정하여 데이터를 필터링하여 조건을 충족하는 데이터만 리스트뷰에 표시되도록 합니다.

## 쿼리 조건&#x20;

리스트뷰를 선택하고 리본 메뉴 모음에서 \[리스트뷰 도구-디자인-> 쿼리 조건 설정]을 선택합니다.

![](<../../.gitbook/assets/image (1568).png>)

또는 리스트뷰를 선택한 후 마우스 오른쪽 버튼을 클릭하고 팝업 메뉴에서 쿼리 조건을 선택합니다.

## 쿼리 조건 설정&#x20;

쿼리 조건 대화 상자에서 쿼리할 데이터 테이블을 선택한 후 새 조건을 클릭하여 쿼리 조건을 추가합니다. 여러 쿼리 조건, 여러 조건 및/또는 관계 연결을 추가할 수 있습니다.

쿼리 조건을 선택하고 \[조건 삭제]를 클릭하여 쿼리 조건을 삭제합니다. 쿼리 조건을 위 또는 아래로 이동하려면 클릭합니다.![](https://help.grapecity.com.cn/download/thumbnails/72354882/image2018-12-18_14-21-41.png?version=1\&modificationDate=1648092569000\&api=v2)

예를 들어 리스트뷰를 선택하고 \[쿼리 조건 설정]를 클릭하고 다음 그림과 같이 \[고객명]이 \[유재석]과 같고 \[완료 여부]를 \[완료됨]으로 쿼리 조건을 추가합니다.

![](<../../.gitbook/assets/image (760).png>)

실행 후 구매자가 유재석이고 완료된 주문만 나오는 것을 확인할 수 있습니다.&#x20;
