# 런타임 중 언어 전환

런타임 시 시스템이 표시하는 언어를 전환하려면 이 명령을 사용하십시오.



셀 리소스를 설정한 후( 자세한 내용은 [다국어](../multi-language.md) 참조 ) "런타임 언어 명령 전환" 명령을 사용하여 버튼을 추가합니다. 런타임 언어 드롭다운 상자에서 런타임 언어를 선택할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (2401).png" alt=""><figcaption></figcaption></figure>

페이지를 실행한 후 버튼을 클릭하면 리소스가 설정된 셀의 언어가 전환된 것을 확인할 수 있습니다.
