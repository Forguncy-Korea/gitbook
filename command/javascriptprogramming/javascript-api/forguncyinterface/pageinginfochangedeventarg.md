# PageingInfoChangedEventArg 인터페이스

리스트뷰 페이징 변경 이벤트에 대한 데이터를 제공합니다.



| 속성                   | 형식     | 설명                                  |
| -------------------- | ------ | ----------------------------------- |
| CurrentPageIndex     | number | <p>0부터 시작하여 현재 페이지의 인덱스입니다.<br></p> |
| MaxRowCountOfOnePage | number | 한 페이지의 최대 행 수입니다.                   |
| TotalRowCount        | number | 페이지 수입니다.                           |
