# CellEvents클래스

### 설명&#x20;

셀에서 지원하는 이벤트입니다.



| 이름                                                        | 트리거 타이밍                                                     |
| --------------------------------------------------------- | ----------------------------------------------------------- |
| [Click](click.md)                                         | 셀을 클릭하면 버튼, 이미지 및 하이퍼링크 셀 유형을 지원하는 이벤트가 트리거됩니다.             |
| [MouseEnter](mouseenter.md)                               | 이 이벤트는 마우스가 셀에 들어갈 때 트리거되며 버튼, 이미지  및 하이퍼링크 셀 유형을 지원합니다.    |
| [MouseLeave](mouseleave.md)                               | 마우스가 셀을 떠날 때 트리거되며 버튼, 이미지 및 하이퍼링크 셀 유형을 지원합니다.             |
| [PivottableClick](pivottableclick.md)                     | 피벗 테이블의 셀을 클릭하면 이벤트가 트리거됩니다.                                |
| [SelectionChanged](../listviewevents/selectionchanged.md) | 셀의 선택 항목이 변경될 때 이벤트가 트리거됩니다. 콤보 상자 및 사용자 선택 상자 셀 유형을 지원합니다. |
| [ValueChanged](valuechanged.md)                           | 셀의 값이 변경될 때 이벤트가 트리거됩니다.                                    |
