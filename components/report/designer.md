# 보고서 디자이너 및 일반적인 작업

## 보고서 디자이너&#x20;

### 보고서 전역 속성 설정&#x20;

디자인 영역의 회색 영역을 클릭하여 디자인 영역이 보고서 요소를 선택하지 않도록 하면 오른쪽의 속성 설정 패널에 전체 보고서의 전역 설정 속성이 표시됩니다.

배경색, 배경 그림, 보고서 페이지의 테두리 두께, 선 유형, 색상 등을 설정할 수 있습니다.

## 일반적인 작업&#x20;

보고서는 여러 보고서 구성 요소로 구성되며 보고서를 만들 때 보고서 컨트롤을 디자인 영역으로 드래그한 다음 요구 사항을 충족하기 위해 설정을 조정해야 합니다. 가장 일반적으로 사용되는 두 가지 기본 작업은 구성 요소 선택 및 데이터 바인딩입니다.

구성 요소를 선택하면 위치 이동, 크기 조정, 속성 설정 등을 수행할 수 있습니다. 구성 요소가 실제 데이터를 표시하도록 하기 전에 데이터를 바인딩할 수 있습니다.

### 구성 요소 선택&#x20;

선택한 구성 요소를 지원하는 세 가지 방법, 다른 구성 요소에서 지원하는 다양한 유형의 구성 요소, 예를 들어 테이블을 예로 들어 보겠습니다.

#### 방법 1. 클릭

테이블에서 셀을 선택한 다음 왼쪽 위 모서리에 나타나는 십자 기호를 클릭하여 전체 테이블을 선택합니다(테이블 및 거리 테이블 이외의 컨트롤을 클릭하여 전체 구성 요소를 선택합니다).

![](<../../.gitbook/assets/image (918).png>)

#### 방법 2. 테이블 선택

마우스를 드래그하여 테이블을 선택합니다.



### 데이터 바인딩&#x20;

보고서에 데이터 원본이 추가된 경우 보고서 구성 요소에 대한 데이터 바인딩을 수행합니다.

세 가지 데이터 바인딩 방법을 지원하며 구성 요소마다 지원되는 다양한 유형이 있으며 다음 표에서는 다음을 예로 들어 보겠습니다.

**방법 1:** 테이블 셀을 선택하고 셀 내의 오른쪽을 클릭하여 바인딩할 데이터 집합의 필드를 선택합니다.![](https://help.grapecity.com.cn/download/thumbnails/56536008/image2018-6-13_15-46-27.png?version=1\&modificationDate=1611038058000\&api=v2)

![](<../../.gitbook/assets/image (1280).png>)

**방법 2:** 데이터 집합 필드를 테이블 셀로 직접 끌어 놓아 데이터를 빠르게 바인딩할 수 있습니다.

**방법 3:** 테이블의 셀을 선택한 후 오른쪽 옵션 설정 패널에서 **데이터** 를 클릭한 후 데이터 또는 표현식을 바인딩합니다.![](https://help.grapecity.com.cn/download/thumbnails/56536008/image2018-6-14_14-4-27.png?version=1\&modificationDate=1611038058000\&api=v2)
