# 논리명령 - 중단명령

포건시의 페이지에서 중단명령을 사용하여 루프명령의 모든 명령을 종료시킬 수 있습니다.

<figure><img src="../../.gitbook/assets/image (2010).png" alt=""><figcaption></figcaption></figure>

이 명령은 루프명령에서만 사용할 수 있습니다. 이 명령을 실행한 후 루프가 종료되고 루프의 후속 명령이 실행됩니다.<br>

루프 종료 명령에 매개변수를 설정할 필요가 없습니다.
