# 데이터 쿼리 조건

데이터 쿼리에는 쿼리 명령 및 테이블 쿼리가 포함됩니다.

* [쿼리 명령](querycmd.md): 쿼리 명령을 사용하여 특정 테이블에 대한 쿼리 조건을 설정하여 테이블에 표시하지 않으려는 레코드를 필터링합니다.

&#x20;    테이블이 로드되면 쿼리 조건에 따라 필터링되어 조건이 충족된 결과를 표시합니다.

* [리스트뷰 쿼리](listviewquery.md) : 테이블에는 테이블의 레코드가 표시되며 모든 레코드가 아닌 특정 기준을 충족하는 레코드만 리스트뷰에 표시하려는 경우 리스트뷰 쿼리를 사용할 수 있습니다.

&#x20;  쿼리 명령이 실행되면 테이블은 쿼리 조건에 따라 데이터를 다시 로드하고 필터링합니다.
