# 이미지 필드

데이터 테이블에 이미지 필드를 추가하여 이미지 데이터를 저장할 수 있습니다.

## 이미지 필드를 추가&#x20;

그림 필드를 추가하는 방법에는 두 가지가 있습니다.

방법 1 데이터 테이블에서 첫 번째 행의 마지막 셀(필드 추가)의 드롭다운 버튼을 클릭하여 드롭다운 목록에서 이미지 유형을 선택합니다.

![](<../../.gitbook/assets/image (1547).png>)

방법 2 데이터 테이블을 열고 리본 메뉴에서 테이블 도구 필드를 선택하고 이미지 유형의 버튼을 클릭합니다.

![](<../../.gitbook/assets/image (1153).png>)

## 이미지 데이터 저장&#x20;

페이지에서 이미지 업로드 셀 유형을 만들고 그림 필드를 바인딩할 수 있습니다. 페이지에서 셀을 클릭하여 이미지를 업로드하고 데이터베이스에 제출하면 실제로 "GUID\_ 파일 이름"으로 저장됩니다.

![](<../../.gitbook/assets/image (1917).png>)

이미지 업로드를 통해 활자 격자 기본 제공 데이터베이스에 업로드된 그림은 기본적으로 \[C:\Users\Public\Documents\ForguncyServer\\_응용 프로그램 이름_\Upload] 디렉토리에 저장됩니다.
