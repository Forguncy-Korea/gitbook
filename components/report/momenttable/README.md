# 테이블릭스(모멘트테이블)를 사용하여 보고서를 디자인

테이블릭스 행과 열로 그룹화된 데이터 요약을 표시하는 데 사용되며, 행과 열은 모두 동적으로 구성된 교차 분석 테이블에 적용됩니다. 가장 일반적인 시나리오는 데이터의 가로 전치 및 데이터의 다차원 교차 분석입니다.

테이블릭스와 테이블의 가장 중요한 차이점은 모멘트 테이블의 행과 열이 동적으로 확장될 수 있고 테이블의 열이 상대적으로 고정되어 있다는 것입니다.

다음 그림은 설계 및 미리 볼 때 모멘트 테이블의 상태를 보여줍니다. 디자인에서 미리 보기까지 모멘트 테이블의 행과 열이 바인딩된 필드의 실제 값에 따라 행과 열 방향으로 동적으로 확장되는 것을 볼 수 있습니다.

## 모멘트 테이블 어셈블리 추가&#x20;

![](<../../../.gitbook/assets/image (2111).png>)

보고서에 데이터 원본이 추가된 경우 모멘트 테이블 어셈블리를 클릭하거나 드래그할 때 디자인 마법사가 나타납니다.

설계 마법사에 대한 자세한 내용은 모멘트 테이블 설계 마법사를 참조하십시오.

설계 마법사를 사용하여 모멘트 테이블을 빠르게 만들거나 설계 마법사의 오른쪽 아래 모서리에 있는 취소 버튼을 클릭하여 빈 모멘트 테이블을 추가한 다음 수동 자유 설계 및 레이아웃을 수행할 수 있습니다

## **토크 미터 리본** <a href="#id-shi-yong-ju-biao-zu-jian-she-ji-bao-biao-3.-ju-biao-gong-neng-qu" id="id-shi-yong-ju-biao-zu-jian-she-ji-bao-biao-3.-ju-biao-gong-neng-qu"></a>

모멘트 테이블의 설계 마법사를 사용하지 않는 경우 보고서에 빈 모멘트 테이블이 직접 추가됩니다. 빈 모멘트 테이블의 기본값은 각각 텍스트 상자가 있는 두 개의 행과 두 개의 열, 4개의 셀입니다.

이 네 개의 셀 행은 모멘트 테이블의 네 가지 기능 영역, 즉 테이블 각도 범위, 열 그룹화 셀, 행 그룹화 셀 및 데이터 범위를 나타냅니다.&#x20;
