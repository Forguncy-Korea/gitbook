# 동시 사용자 제어

포건시는 세션 제어를 지원합니다. 포건시 서버측에서 서버 제어를 활성화할 수 있습니다. 세션 제어가 활성화된후 사용자가 설정된 세션 제한 시간 내에 애플리케이션을 작동하지 않으면 자동으로 연결이 끊어지고 애플리케이션이 로우아웃됩니다.&#x20;





### 제한 시간 설정&#x20;

![](https://help.grapecity.com.cn/download/thumbnails/80953052/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1673923283000\&api=v2)    "로그 및   모니터링-> 동시 라이선스관리" 페이지에서 "제한 시간 설정 변경을클릭하고 팝업 페이지에서 시간 초과 설정을 수정하십시오.\
<br>

<figure><img src="../../.gitbook/assets/image (2591).png" alt=""><figcaption></figcaption></figure>

* 비활성화: 이 항목을 선택하면 세션 제어가 활성화되지 않음을 의미하며, "사용자가 페이지를 닫은 후 로그아웃하지 않은 경우 연결 유지 시간" 및 " 사용자가 페이지를 닫은 후 연결 유지 시간 "을 설정할 수 없습니다. 작동하지 않습니다." 반응이 없습니다.
* 활성화됨(로그아웃하지 않고 앱 연결 끊기): 이 옵션을 선택하면 "사용자가 작업하지 않을 때 연결 유지 시간"을 설정할 수 있으며, 사용자가 설정한 시간 내에 페이지를 작업하지 않으면 브라우저가 자동으로 해당 페이지로 이동합니다. 연결 해제 페이지에서 페이지를 다시 로드해야 합니다.
* 활성화(세션활성화): 이 항목을 선택하면 세션 제어가 활성화됨을 의미하며, "사용자가 로그아웃하지 않을 때 페이지를 닫은 후 연결 유지 시간"과 "사용자가 조작하지 않을 때 연결 유지 시간"을 설정할 수 있습니다. " .

![](https://help.grapecity.com.cn/download/thumbnails/80953052/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1673923283000\&api=v2)  사용자 로그인 상태를 관리합니다. 모든 사용자는 "로그및모니터링-> 동시 사용자 설정"  페이지에 나열되며, 관리자는 "퇴출"을 클릭하여 로그인한 사용자를 강제로 추방할 수 있습니다. 사용자가 퇴출된에후는 응용 프로그램에 다시 로그인해야 합니다.<br>
