# 페이지 내 탭 방식 셀

페이지 내  탭 방식사셀을용하면 일반 페이지에 적용된 마스터 페이지의 페이지 자리 표시자와 유사하게 한 페이지에서 다른 페이지를 하위 페이지로 사용할 수 있습니다. 탭 셀은 하위 페이지를 탭으로 표시합니다.

탭 제목은 하위 페이지의 제목을 사용하거나 제목이 설정되지 않은 경우 페이지 이름을 사용합니다.

페이지에서 영역을 선택하고 해당 셀 유형을 탭 셀로 설정하십시오.이 영역은 하위 페이지를 표시하는 데 사용됩니다.서브 페이지의 크기에 따라 영역의 크기를 조정하십시오.

## 설정&#x20;

페이지 내 탭 방식 셀을  선택하고 속성 설정 영역에서 "셀 유형" 탭 페이지를 선택합니다.

| 설정             | 설명                                                                                                                                                                                                                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 서브페이지          | <p>설정 탭의 하위 페이지. 탭을 늘리거나 줄이려면 "+" 또는 "-"를 클릭하고 탭의 하위 페이지를 설정하려면 드롭다운 화살표를 클릭합니다.</p><p>동일한 유형의 페이지만 선택할 수 있습니다. 즉, 일반 페이지는 하위 페이지로만 설정할 수 있고, 모바일 페이지는 모바일 페이지의 하위 페이지로만 설정할 수 있으며, 템플릿 페이지나 내장 페이지는 하위 페이지로 설정할 수 없습니다.</p><p>서브 페이지 설정 후 탭 셀을 더블 클릭하면 기본으로 설정된 서브 페이지가 편집용으로 바로 열립니다.</p> |
| 활성 탭 번호        | <p>기본적으로 선택되는 탭 번호를 설정합니다.</p><p>탭은 1부터 시작하여 번호가 매겨지며 탭 수보다 큰 숫자를 입력하면 기본 탭은 첫 번째 탭입니다.</p>                                                                                                                                                                                                 |
| 하위 페이지 오버플로 모드 | <ul><li>오버플로: 하위 페이지를 완전히 표시할 수 있을 때까지 탭이 확장됩니다.</li><li>스크롤: 탭에 스크롤 막대가 나타납니다.</li><li>자르기: 하위 페이지에서 탭을 초과하는 부분은 잘려서 표시되지 않습니다.</li></ul><p>하위 페이지를 선택한 후 하위 페이지의 오버플로 모드를 설정할 수 있으며 이 옵션을 사용하여 하위 페이지 표시 방법을 설정할 수 있습니다. 세 가지 모드가 있습니다.</p>                                               |
| 탭 숨기기          | 탭의 제목은 숨겨져 있으며 "사용 가능한 작업"의 "옵션 설정"을 통해 표시되는 페이지를 제어할 수 있습니다.                                                                                                                                                                                                                               |
| 스타일 설정         | 테두리 및 탭 제목의 색상 스타일을 설정합니다. 레이블 제목의 글꼴, 글꼴 스타일과 같은 기타 스타일 설정은 셀의 설정을 사용합니다.                                                                                                                                                                                                                  |
| 지연 로드          | 이 항목을 체크하면 현재 선택한 페이지가 동적으로 로드되고, 체크하지 않으면 탭의 모든 페이지가 로드됩니다.                                                                                                                                                                                                                                |
| 관련 명령 빠른 생성    | 탭에 사용할 수 있는 작업은 선택 설정, 탭 숨기기 및 탭 표시입니다. 사용 가능한 작업을 페이지로 끌어다 놓아 셀 작업 명령을 생성합니다.                                                                                                                                                                                                              |
