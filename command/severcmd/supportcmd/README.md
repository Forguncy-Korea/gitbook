# 서버단 명령에서 지원하는 명령

서버단 명령을 편집할 때 서버단에서 실행되는 명령을 설정해야 합니다. 서버단 명령에서 지원하는 명령에 몇 가지 새로운 명령이 추가되었으며 이 섹션에서는 모든 서버 쪽 명령에 설정할 수 있는 명령에 대해 설명합니다.



## 서버단 명령에서 지원하는 명령&#x20;

| 명령                    | 설명                                                                                                                                                          |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 데이터 테이블 업데이트하기        | 데이터 테이블에 대한 직접 업데이트, 추가 및 삭제 작업을 수행할 수 있습니다. 작업 유형을 선택한 후 대상 테이블을 선택하여 대상 데이터 테이블에 대한 작업을 수행합니다.                                                            |
| 페이지를 Excel로 내보내기      | 페이지를 Excel로 내보내기 명령을 사용하여 전체 페이지의 내용을 Excel로 내보낼 수 있습니다.                                                                                                    |
| 페이지를 PDF로 내보내기        | 페이지를 PDF로 내보내기 명령을 사용하면 전체 페이지의 내용을 PDF로 내보낸 다음 인쇄할 수 있습니다.                                                                                                 |
| 조건문 만들기               | 조건 명령을 사용하면 If, ElseIf 및 Else를 사용하여 조건의 다른 분기를 구성하고, 조건 집합을 설정하고, 조건이 충족되면 하나 또는 하나의 명령 집합을 실행하는 조건에 따라 다른 명령을 설정할 수 있습니다. 그렇지 않은 경우 다른 명령 또는 명령 집합이 실행됩니다. |
| 메일 전송 명령 생성하기         | 메일 보내기 명령을 사용하여 특정 사용자 또는 전자 메일 주소로 메시지를 보냅니다.                                                                                                              |
| 루프 명령 만들기             | 루프 명령을 사용하여 특정 루프 방식으로 하나 또는 하나의 하위 명령 집합을 반복합니다.                                                                                                           |
| 프로그램 실행               | 프로그램 호출 명령을 사용하여 외부 프로그램을 호출할 수 있습니다.                                                                                                                       |
| 반환 명령 생성하기            | 서버 쪽 명령을 실행할 때 오류가 발생할 수 있으며 반환 명령을 사용하여 서버 쪽 명령의 실행을 종료하고 오류 코드 및 오류 메시지를 반환할 수 있습니다.                                                                      |
| 서버단 명령 호출             | 서버 쪽 명령을 호출합니다.                                                                                                                                             |
| 트랜잭션 생성하기             | 트랜잭션 명령의 모든 데이터베이스 작업이 트랜잭션에서 실행되고 트랜잭션에 대한 격리 수준을 지정할 수 있는 트랜잭션 명령을 지원합니다.                                                                                 |
| CSV 가져오기 / 내보내기       | 서비스 측에서 CSV 파일을 가져오고 내보냅니다.                                                                                                                                 |
| 파라미터 설정               | 변수 이름 및 변수 값을 포함하여 서비스 측면에서 변수를 설정합니다.                                                                                                                      |
| 저장프로시저 호출하기           | MySQL 및 SQL Server 데이터베이스에 대한 저장 프로시저 호출을 지원합니다.                                                                                                            |
| 페이지를 HTML로 내보내기       | 페이지를 HTML로 내보내기 명령을 사용하여 페이지를 HTML 문자열로 변환하고 결과를 매개 변수에 저장합니다.                                                                                              |
| 예외처러 명령               | 지정된 상황에서 오류를 처리한 다음 오류 상황에 따라 다음 단계를 결정합니다.                                                                                                                 |
| SQL 명령 실행             | 서비스 측에서 SQL 명령을 실행하고, SQL 문을 통해 데이터베이스를 조작하고, 저장 프로시저 작성 및 호출, 임시 테이블 만들기, 테이블 인덱스 만들기 등과 같은 복잡한 기능을 수행합니다.                                                 |
| 중단명령                  | 루프 명령 목록의 모든 명령을 종료합니다.                                                                                                                                     |
| 서버알림                  | 서버 알림 명령에는 서버 알림 명령 보내기, 그룹 가입, 그룹 탈퇴 및 사용자 가입 서버 알림 상태 얻기가 포함됩니다.                                                                                          |
| 서버 가져오기 내보내기 Excel명령  | 서버 명령에서 서버 가져오기 및 내보내기 Excel 명령을 사용하여 서버에서 Excel 데이터를 가져오거나 내보낼 수 있습니다.                                                                                     |
