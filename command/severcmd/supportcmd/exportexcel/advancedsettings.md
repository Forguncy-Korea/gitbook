# 고급설정

서버 측 명령에서 Excel로 페이지 내보내기 명령을 사용하여 서버 측에서 페이지를 내보낼 수 있습니다.

Excel 명령으로 내보내기 페이지에서 고급 설정을 지정하고 내보낸 파일의 비밀번호를 설정할 수도 있습니다. 내보낸 페이지가 내보내기를 위해 페이지에 입력한 값을 서버에 전달해야 하는 경우 내보내기 페이지의 정보를 설정해야 합니다.

### 암호화 비밀번호 설정 <a href="#she-zhi-jia-mi-mi-ma" id="she-zhi-jia-mi-mi-ma"></a>

내보낸 파일을 암호화하려면 다음 텍스트 상자에 암호화 비밀번호를 설정하세요. 비밀번호는 직접 설정하거나 매개변수로 설정할 수 있습니다.
