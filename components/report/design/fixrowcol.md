# 행/열을 고정

페이지를 매김하지 않고 미리 볼 때 테이블 크기가 브라우저 창을 초과하는 경우 페이지를 스크롤하여 미리 보기를 수행해야 합니다. 그러나 페이지를 스크롤할 때 테이블 머리글도 스크롤되어 현재 셀의 제목이 어떤 행인지 또는 어떤 열인지 구분하기가 어렵습니다.

미리 볼 때 고정된 행이나 열이 스크롤 막대와 함께 스크롤되지 않도록 행과 행을 고정하는 기능을 보고서에 제공합니다.

전체 테이블을 선택하여 오른쪽의 속성 설정 패널에서 **고정된 행 수**와 **고정된 열 수를** 찾습니다.
