# 행작동

리스트뷰 업데이트 명령만들기에서 작업 유형을 선택하여 작업 행 범주 아래에 편집, 추가, 삽입 및 삭제의 네가지 작업 유형이 있습니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (1235).png" alt=""><figcaption></figcaption></figure>

### 편집&#x20;

작업 유형이 "편집"인 경우 리스트뷰의 데이터를 업데이트할 수 있지만 데이터베이스에 즉시 제출되지는 않습니다.

<figure><img src="../../../.gitbook/assets/image (923).png" alt=""><figcaption></figcaption></figure>

다음 설정이 필요합니다.

* 대상 리스트뷰:   페이지에서 리스트뷰를 선택해야 하며 리스트뷰이명령 작동이실행될 때 리스트뷰 데이터가 업데이트됩니다.
* 열   및 값: 리스트뷰의 데이터를 업데이트할 열과 데이터를 설정합니다.
  * 열: 수식을 사용하여 리스트뷰 템플릿 행의 셀을 참조하거나 열 이름을 입력할 수 있습니다.
  * 데이터: 업데이트된 칼럼의 데이터를 설정합니다. 데이터는 고정 값, 수식 또는 키워드일 수 있습니다.
* 업데이트  할:  행 업데이트할 기록을 설정합니다. 리스트뷰에서 선택한 행, 행이  행이 리스트뷰의 조건을 충족함, 리스트뷰에서 모든 행, 리스트뷰의 현재 행을  선택할 수 있습니다. 그 중 리스트뷰에서 선택한 행, 행이  행이 리스트뷰의 조건을 충족함, 리스트뷰에서 모든 행에 대한 설정은 데이터 테이블 연산 명령어에서 갱신 레코드와 동일하다. 고급 설정 - 특정 레코드를 작동합니다 . 리스트뷰의 현재 행은 리스트뷰에서 현재 선택된 셀의 행을 참조합니다.

### 추가&#x20;

작업 유형이 "추가"인 경우 데이터를 리스트뷰에 추가할 수 있지만 데이터베이스에 즉시 반영되지는 않습니다.

<figure><img src="../../../.gitbook/assets/image (2090).png" alt=""><figcaption></figcaption></figure>

다음 설정이 필요합니다.

* 대상  리스트뷰: 페이지에서 리스트뷰를 선택하고 리스트뷰 명령  작동을 실행할 때 리스트뷰에 데이터를 추가해야 합니다.
* 열   및 값: 리스트뷰에 데이터를 추가할 열과 데이터를 설정합니다.
  * 열: 수식을 사용하여 리스트뷰 템플릿 행의 셀을 참조하거나 열 이름을 입력할 수 있습니다.
  * 데이터: 추가된 칼럼의 데이터를 설정합니다. 데이터는 고정 값, 수식 또는 키워드일 수 있습니다.



### 삽입&#x20;

작업 유형이 "삽입"인 경우 리스트뷰에 행을 삽입할 수 있지만 데이터베이스에 즉시 제출되지는 않습니다.

<figure><img src="../../../.gitbook/assets/image (2245).png" alt=""><figcaption></figcaption></figure>

다음 설정이 필요합니다.

* 대상  리스트뷰: 페이지에서 리스트뷰를 선택해야 하며 리스트뷰  명령 작동이   실행될 때 리스트뷰의 데이터가 업데이트됩니다.
* 열   및 값: 리스트뷰에 삽입할 컬럼 및 데이터를 설정합니다.
  * 열: 수식을 사용하여 리스트뷰 템플릿 행의 셀을 참조하거나 열 이름을 입력할 수 있습니다.
  * 값: 삽입된 컬럼의 데이터를 설정합니다. 데이터는 고정 값, 수식 또는 키워드일 수 있습니다.
* 위치  삽입 : 현재 행 위 또는 현재 행 아래에 삽입하도록 선택할 수 있습니다.

### 삭제&#x20;

작동유형이 "삭제"인 경우 리스트뷰의 데이터를 삭제할 수 있지만 데이터베이스에 즉시 제출되지는 않습니다.

<figure><img src="../../../.gitbook/assets/image (1637).png" alt=""><figcaption></figcaption></figure>

다음 설정이 필요합니다.

* 대상:리스트뷰:   페이지에서 리스트뷰를 선택해야 하며 리스트뷰 작업 명령이 실행되면 리스트뷰의 데이터가 삭제됩니다.
* 업데이트  할 행: 삭제할 기록을 설정합니다. 리스트뷰에서 선택한 행, 행이  행이 리스트뷰의 조건을 충족함, 리스트뷰에서 모든 행, 리스트뷰의 현재 행을 선택할 수 있습니다. 이 설정은 데이터 테이블 작업 명령의 업데이트 레코드와 동일하며 자세한 내용은 [고급 설정 - 작업별 레코드](../update/recordbyjob.md)를 참조하십시오 .
