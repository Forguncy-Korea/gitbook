## **2. 페이지를 PDF로 인쇄/내보내기** <a href="#id-da-yin-ming-ling-da-yin-dao-chu-ye-mian-dao-pdf2.-da-yin-dao-chu-ye-mian-dao-pdf" id="id-da-yin-ming-ling-da-yin-dao-chu-ye-mian-dao-pdf2.-da-yin-dao-chu-ye-mian-dao-pdf"></a>

PDF로 페이지 내보내기 명령의 사용에 대한 자세한 내용은 인쇄를 참조하십시오.

{% hint style="info" %}
페이지에 테이블이 있는 경우 테이블이 내보낸 테이블에 영향을 주는 데이터를 요청 시 로드할 수 있는지 여부입니다.

* 테이블에 로드 레코드 행 수가 설정되어 있고 요청 시 로드가 활성화되지 않은 경우. 이 시점에서 테이블에는 로드된 레코드 행 수에 대한 일부 데이터만 표시할 수 있으며 표시된 데이터만 내보낼 수 있습니다.
* 테이블에 로드 레코드 행 수가 설정되어 있고 요청 시 로드가 활성화된 경우. 테이블에는 모든 데이터가 표시되고 내보낼 때 내보낼 수 있습니다.
{% endhint %}
