# 인쇄 영역

포건시에서 지정된 페이지 또는 페이지의 지정된 영역을 인쇄할 수 있습니다. 인쇄 범위는 인쇄를 지정하는 하나 이상의 셀 범위입니다. 인쇄 영역을 정의한 다음 페이지를 인쇄할 때 지정된 셀 범위만 인쇄됩니다.

필요에 따라 셀을 추가하여 인쇄 영역을 확장하거나 인쇄 영역을 지워 전체 페이지를 인쇄할 수 있습니다. 페이지에는 여러 인쇄 영역이 있을 수 있으며 각 인쇄 영역은 별도의 페이지로 인쇄됩니다.

## 페이지 인쇄&#x20;

기본적으로 인쇄 영역이 설정되지 않은 경우 전체 페이지를 인쇄할 수 있습니다.<br>

인쇄된 페이지에 테이블이 포함되어 있고 테이블의 레코드 행 수가 페이지에 설정된 테이블 행 수보다 크면 인쇄 시 테이블의 모든 데이터가 자동으로 확장되고 인쇄됩니다.



![](<../../.gitbook/assets/image (2024).png>)

실행 후 페이지의 테이블 데이터가 모두 로드되지만 리스트 크기 제한으로 인해 완전히 표시되지 않습니다. \[인쇄] 버튼을 클릭하면 리스트뷰가 PDF로 내보내지고 리스트뷰가 자동으로 확장되어 리스트뷰의 모든 데이터가 인쇄됩니다.

![](<../../.gitbook/assets/image (2023).png>)

### 인쇄 영역 설정 <a href="#she-zhi-da-yin-qu-yu" id="she-zhi-da-yin-qu-yu"></a>

페이지의 특정 영역을 인쇄하려는 경우 인쇄 영역을 설정할 수 있습니다. 인쇄 영역은 인쇄를 위해 지정된 하나 이상의 셀 범위입니다. 인쇄 영역을 정의한 후 페이지를 인쇄하면 지정된 셀 범위만 인쇄됩니다.

필요에 따라 셀을 추가하여 인쇄 영역을 확장하거나 인쇄 영역을 지워 전체 페이지를 인쇄할 수 있습니다. 페이지에는 여러 인쇄 영역이 있을 수 있으며 각 인쇄 영역은 별도의 페이지로 인쇄됩니다.

예를 들어 페이지에서 테이블을 선택한 다음 리본 메뉴 표시줄에서 "인쇄->인쇄 영역"을 선택하고 "인쇄 영역 설정"을 클릭합니다.
