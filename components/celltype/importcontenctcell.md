# 페이지 내 컨텐츠가 포함된 셀

일반 페이지에 적용된 마스터 페이지의 페이지 자리 표시자와 유사하게 페이지가 다른 페이지를 하위 페이지로 사용할 수 있습니다.

페이지에서 영역을 선택하고 해당 셀 유형을 페이지 컨테이너 셀로 설정하십시오.이 영역은 하위 페이지를 표시하는 데 사용됩니다.서브 페이지의 크기에 따라 영역의 크기를 조정하십시오.

<figure><img src="../../.gitbook/assets/image (1609).png" alt=""><figcaption></figcaption></figure>

## 설정&#x20;

페이지 내 컨텐츠가 포함된 셀을 선택하고 속성 설정 영역에서 "셀 설정" 탭을 선택합니다. 설정 지침은 아래 표를 참조하십시오.

<table><thead><tr><th width="272">설정 </th><th>설명 </th></tr></thead><tbody><tr><td>하위페이지 </td><td><p>페이지 컨테이너에 표시되는 페이지를 설정하려면 같은 종류의 페이지만 선택 가능하며 기본제공페이지는 선택할 수 없습니다. 모바일 페이지만 하위 페이지로 선택할 수 있습니다.</p><p>서브 페이지를 설정한 후 페이지 컨테이너 셀을 더블 클릭하여 편집할 서브 페이지를 바로 엽니다.</p></td></tr><tr><td>하위페이지오버플로 모드 </td><td><p>하위 페이지를 선택한 후 하위 페이지의 오버플로 모드를 설정할 수 있으며 이 옵션을 사용하여 하위 페이지 표시 방법을 설정할 수 있습니다. 세 가지 모드가 있습니다.</p><p> • 오버플로: 하위 페이지가 완전히 표시될 때까지 페이지 컨테이너가 확장됩니다.</p><p> • 스크롤: 페이지 컨테이너에 스크롤 막대가 나타납니다.</p><p> • 잘라내기: 하위 페이지에서 페이지 컨테이너를 초과하는 부분은 클리핑되어 표시되지 않습니다.</p></td></tr></tbody></table>

**하위 페이지 오버플로 모드**

오버플로, 스크롤 및 잘라내기의 세 가지 하위 페이지 오버플로 모드가 있습니다.

예를 들어 상위 페이지에서 페이지 컨테이너를 7행 7열로 설정하고 테두리를 빨간색 테두리로 설정하고 하위 페이지를 10행 10열로 설정합니다.

* **오버플로** : 페이지 컨테이너가 확장되어 하위 페이지를 완전히 표시합니다. 아래 그림과 같이 하위 페이지 오버플로 모드를 "오버플로"로 설정합니다.

실행 후 하위 페이지가 페이지 컨테이너를 오버플로하는 것을 볼 수 있습니다

* **스크롤** : 페이지 컨테이너에 스크롤 막대가 나타납니다. 아래 그림과 같이 하위 페이지 오버플로 모드를 "스크롤"로 설정합니다.
