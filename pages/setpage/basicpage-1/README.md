# 기본 제공 페이지

새 빈 프로젝트를 만들면 기본 제공 페이지 및 기본 제공 모바일 페이지를 포함한 일부 기본 페이지를 볼 수 있습니다.&#x20;

기본 제공 페이지의 인터페이스와 모양을 수정할 수 있지만 로그인 페이지의 사용자 이름, 암호 및 로그인 버튼과 같은 특수 기능이 있는 텍스트 상자, 버튼 또는 하이퍼링크는 삭제할 수 없으며 모양만 수정할 수 있습니다.

기본제공페이지는 일반 페이지와 모바일페이지로 나뉩니다.&#x20;

## 기본 제공 페이지&#x20;

기본 제공 페이지에는 기본 제공 모바일 페이지, 기본 제공 일반 페이지 및 기본 제공 마스터 페이지가 포함됩니다. 내장된 마스터 페이지는 포건시의   기본제공페이지에 적용됩니다. 내장된 마스터 페이지의 이름은 수정하거나 삭제할 수 없습니다.

인증 모드가 다른 경우 내장 페이지가 다르게 표시됩니다.

*   인증 모드가 일반 인증 또는 타사 사용자 통합인 경우 기본 제공 페이지는 다음과 같습니다.<br>

일반 인증에서 Windows 도메인 인증으로 전환하면 불필요한 내장 페이지가 자동으로 제거되고, Windows 도메인 인증에서 일반 인증으로 전환하면 제거된 내장 페이지가 자동으로 다시 추가됩니다. 인증 모드를 전환할 때 이전에 사용자 정의된 수정 사항은 저장되지 않습니다.
