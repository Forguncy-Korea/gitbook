# 실행한 쿼리 결과를 상위 페이지에 전달

팝업 페이지 닫기 명령을 설정하면 쿼리 결과를 상위 페이지로 전달하도록 설정된 항목이 있으며, 이 옵션을 선택하면 팝업 페이지를 닫은 다음 상위 페이지로 돌아가면 팝업 페이지의 쿼리 결과도 상위 페이지로 전달됩니다.

쿼리 결과를 전달하는 것은 쿼리된 레코드입니다.



상위 페이지에 레코드가 있고 기록된 쿼리를 위한 페이지가 팝업되면 팝업 페이지 명령을 닫는 옵션에서 "실행한 쿼리 결과를 상위 페이지에 전달"을 선택하면 쿼리 결과가 상위 페이지에 표시됩니다.

예를 들어 주문 목록 페이지에서 쿼리 페이지를 팝업하는 버튼을 설정하고 쿼리 페이지의 확인 버튼에서 팝업 페이지 닫기 명령을 설정하고 다음 그림과 같이 쿼리 결과를 상위 페이지로 전달을 선택합니다.

![](<../../../.gitbook/assets/image (1041).png>)

실행 후 주문 목록 페이지에서 쿼리 버튼을 클릭하여 쿼리 페이지를 팝업합니다. 쿼리 페이지에서 쿼리한 후 확인을 클릭하면 쿼리 결과가 상위 페이지에 표시됩니다.
