# 페이지 스트리밍 레이아웃

포건시의 페이지에는 행 높이와 열 너비의 세 가지 모드가 있습니다: 고정 모드, 적응 모드 및 범위 모드, 적응 모드 또는 범위 모드로 행 높이 및 열 너비 조정 모드를 설정하여 페이지가 스트리밍 레이아웃을 렌더링하고 페이지 레이아웃을 보다 유연하게 만들 수 있습니다.

기본값은 고정 모드입니다. 고정 모드에서 행 높이와 열 너비를 픽셀 단위로 고정된 크기로 설정합니다. 이 모드에서는 페이지의 레이아웃이 고정됩니다.

다음은 적응형 및 범위 모드를 사용하여 페이지를 스트리밍 레이아웃으로 만드는 방법에 대해 자세히 설명합니다.

## 적응 모드&#x20;

적응 모드에서는 콘텐츠에 따라 페이지가 자동으로 확장됩니다. 열 너비를 적응 모드로 설정하면 너비가 자동으로 확장됩니다. 행 높이를 가변 모드로 설정하면 높이가 자동으로 확장됩니다.

행 높이를 적응 모드로 설정하는 예를 들어 보겠습니다.

페이지에서 셀 범위를 텍스트 상자로 설정하고 텍스트 상자 줄 바꿈을 설정한 다음 해당 범위에서 하나 이상의 행을 선택하여 행 높이를 설정합니다.

행 헤더를 마우스 오른쪽 버튼을 클릭하고 행 높이를 선택하면 열 너비 설정 대화 상자가 나타나고 \[행 높이]를 선택합니다.

적응 모드로 설정된 행 인덱스는 빨간색으로 바뀝니다.

<figure><img src="../../.gitbook/assets/image (561).png" alt=""><figcaption></figcaption></figure>

실행 후 텍스트 상자에 내용을 입력하면 콘텐츠의 너비가 텍스트 상자의 높이를 초과하면 자동으로 확장됩니다.

## 범위 모드&#x20;

행 높이/열 너비를 범위 모드로 설정하려면 최소값과 최대값을 설정해야 합니다. 최소값은 픽셀이고 최대값은 픽셀 또는 백분율입니다.

* 최대값 픽셀 설정: 페이지에 충분한 공간이 있는 경우 행 높이/열 너비가 증가함에 따라 최대값까지 페이지가 부드러워집니다.
* 최대값을 백분율로 설정: 행 높이/열 너비는 페이지의 나머지 공간을 모두 차지하며 나머지 공간은 백분율로 할당됩니다.

예를 들어 열 A의 열 너비를 범위 모드로 설정하고 최소값은 0픽셀이고 최대값은 1입니다.

<figure><img src="../../.gitbook/assets/image (330).png" alt=""><figcaption></figcaption></figure>

B 열 너비는 기본 고정 모드, 20픽셀입니다.

C 열 너비를 범위 모드로 설정하고 최소값은 0픽셀, 최대값은 3입니다.

범위 모드로 설정된 열 인덱스는 보라색으로 바뀝니다.

<figure><img src="../../.gitbook/assets/image (1858).png" alt=""><figcaption></figcaption></figure>

A열과 C열은 각각 전체 페이지의 나머지 공간의 1/4과 3/4을 차지합니다.
