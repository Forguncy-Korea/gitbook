# 로그 및 모니터링

서버단에서는 로그 보기 및 모니터링, 사용자 로그 관리, 애플리케이션과 해당 시스템의 CPU 및 메모리 사용량을 모니터링할 수 있습니다.

포건시 서버 관리자 에서 "로그 및 모니터링"을 선택하면 로그 및 모니터링 페이지로 진입합니다.
