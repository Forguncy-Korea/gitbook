# 논리 명령 - 루프 명령

루프 명령을 사용하여 특정 루프 방식으로 하나 또는 하나의 하위 명령 집합을 반복합니다. 루프는 지정된 횟수만큼, 리스트로, 셀별로 반복됩니다.

![](<../../../.gitbook/assets/image (2022).png>)

## 루프 방법&#x20;

루프에는 지정된 횟수만큼, 리스트별로, 셀별로 반복하는 세 가지 방법이 있습니다.

| 설정           | 설명                                                                                                                                                                                                                                                                                                                                                                                        |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 반복 횟수        | <p>명령을 반복하는 횟수를 설정합니다.</p><p>횟수: 고정 값, 수식 또는 지정된 셀에 대해 하나 또는 하나의 셀 범위를 선택하려면 클릭합니다.<img src="https://help.grapecity.com.cn/download/thumbnails/72354342/image2018-2-1_16-0-33.png?version=1&#x26;modificationDate=1648092561000&#x26;api=v2" alt="">                                                                          * 횟수가 유효하지 않은 경우( 즉, 0보다 큰 숫자가 아닌 경우 하위 명령은 실행되지 않습니다.)</p> |
| 리스트뷰에 따라 루프  | <p>브라우저에서 테이블의 행 수 또는 선택한 행 수인 지정된 테이블로 반복합니다. 루프 중에 레코드 수가 변경되면 루프 수가 변경됩니다.</p><p>하위 명령에 테이블 템플릿 행을 참조하는 수식이 있는 셀이 있는 경우 수식은 첫 번째 루프에서 첫 번째 행의 셀을 참조하고 두 번째 루프에서 두 번째 행의 셀을 참조합니다.</p>                                                                                                                                                                                                  |
| 셀에 따라 루프     | <p>지정된 기준 셀로 반복되며 루프 수는 기준 셀 수입니다.</p><p>하위 명령에서 수식이 상대 참조를 사용하는 경우 루프 중에 수식이 참조하는 셀이 기준 셀로 오프셋됩니다.                                                           *셀별로 순환할 때 하위 명령에 테이블 템플릿 행의 셀을 참조하는 수식이 있는 경우 절대 참조를 사용합니다. 루프 중에 참조 오프셋으로 인해 테이블에서 선택한 행의 값을 얻을 수 없습니다.</p>                                                                                                               |

## 루프 하위 명령&#x20;

루프를 설정한 후에는 하위 명령도 설정해야 합니다. 루프 명령 아래의 빈 명령을 클릭하여 루프의 하위 명령을 추가하여 여러 하위 명령을 추가합니다.
