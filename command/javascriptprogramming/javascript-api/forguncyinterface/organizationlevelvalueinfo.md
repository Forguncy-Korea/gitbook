# OrganizationLevelValueInfo 인터페이스

사용자의 조직 수준 정보입니다. getUserInfo 메서드에 사용됩니다.



| 속성                    | 형식     | 설명                    |
| --------------------- | ------ | --------------------- |
| OrganizationLevelName | string | 조직 수준 이름입니다.          |
| Value                 | string | 조직 수준에서 조직 노드의 이름입니다. |
