# 이미지 목록 인쇄

포건시에서 지정된 페이지를 인쇄하고 페이지에 이미 목록이 포함된 경우 인쇄할 수 있습니다.

기본적으로 인쇄 영역이 설정되지 않은 경우 전체 페이지를 인쇄할 수 있습니다.<br>

인쇄된 페이지에 그래픽 목록이 포함된 경우 인쇄 버튼을 설정하고 "페이지 PDF로 내보내기" 명령으로 설정합니다.

![](<../../.gitbook/assets/image (1119).png>)

\[인쇄] 버튼을 클릭하면 페이지가 PDF로 내보내지고 그래픽 목록이 자동으로 확장되어 이미지 목록의 모든 데이터가 인쇄됩니다.

![](<../../.gitbook/assets/image (1039).png>)



주목해야 할 점은 다음과 같습니다.

* 이미지 목록을 인쇄 영역으로 설정하면 가로 및 흐름 정렬의 N자 배열로 흐르는 경우 인쇄된 그래픽 목록이 자동으로 확장되지 않으므로 모든 데이터를 인쇄할 수 없습니다.
* 이미지 목록 템플릿의 배경 그림과 차트를 인쇄할 수 없습니다.
* 이미지 목록에 데이터가 없는 경우 데이터가 없을 때 표시되는 내용을 인쇄할 수 없습니다.
