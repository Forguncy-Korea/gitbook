# 서버 알림 권한

서버단 알림을 생성할 때 권한을 설정할 수 있습니다. 즉, 구독을 설정할 수 있습니다.이 서버 알림의 역할입니다.

### 서버 알림 권한 설정 <a href="#she-zhi-fu-wu-duan-tong-zhi-quan-xian" id="she-zhi-fu-wu-duan-tong-zhi-quan-xian"></a>

일반 설정 탭에서 "권한 설정"의 "로그인 사용자" 하이퍼링크를 클릭합니다.

<figure><img src="../../../../.gitbook/assets/image (253).png" alt=""><figcaption></figcaption></figure>

팝업 "역할 선택" 대화 상자에 현재 애플리케이션의 모든 역할이 나열됩니다. 하나 이상의 역할을 선택한 후 확인을 클릭하여 대화 상자를 닫습니다.
