# 보고서 매개변수

보고서 매개 변수는 보고서에서 가장 일반적으로 사용되는 데이터 필터링 소스이며 보고서 매개 변수를 추가한 후 보고서를 미리 볼 때 매개 변수 입력 상자가 나타나고 사용자에게 매개 변수 값을 입력하라는 메시지가 표시됩니다.

이 섹션에서는 보고서 매개 변수를 추가하는 프로세스에 대해 설명합니다.

![](https://help.grapecity.com.cn/download/thumbnails/56537582/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1615882811000\&api=v2) 보고서 디자이너의 오른쪽에서 데이터 바인딩 패널을 연 후 다음 그림과 같이 보고서 매개 변수 영역의 추가를 클릭합니다.

![](<../../../.gitbook/assets/image (1553).png>)

![](https://help.grapecity.com.cn/download/thumbnails/56537582/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1615883436000\&api=v2)  보고서 매개 변수 목록에 나타나는 새 항목인 보고서 매개 변수 1을 클릭하면 보고서 매개 변수에 대한 자세한 설정 정보가 패널에 표시됩니다.



![](<../../../.gitbook/assets/image (1465).png>)

보고서 매개 변수 편집 패널의 설정 항목은 다음과 같습니다.

* **매개 변수 이름:** 매개 변수 인식에 사용할 보고서 매개 변수의 이름을 설정합니다.
* **프롬프트 텍스트:** 보고서를 미리 볼 때 매개 변수 패널의 프롬프트입니다.
* **데이터 형식:** 상황에 따라 매개 변수 값의 데이터 형식을 선택합니다. 부울, 날짜, 날짜 및 시간, 정수, 부동 소수점, 문자열을 지원합니다.\
  데이터 형식을 날짜 또는 날짜 시간 유형으로 선택하면 매개 변수 패널 입력 상자를 클릭하면 캘린더 컨트롤이 자동으로 로드되므로 사용자 입력이 더 편리합니다.
* **다중 값:** 보고서 매개 변수 값이 다중 선택인 경우 다중 값 데이터 기능을 켭니다. 다중 값 데이터의 특정 사용.
* **숨김:** 설정하면 매개변수 패널이 숨겨지며 일반적으로 매개변수 자동 값 전달과 같이 사용자 입력이 필요하지 않은 장면에 사용됩니다. 숨기기를 켤 때 기본 데이터를 설정해야 합니다.
* **Null 값 허용:** 이 인수는 null일 수 있습니다. IsNothing() 표현식을 사용하여 동적 데이터 필터링을 위해 비어 있는지 여부를 확인할 수 있습니다.
* **여러 줄:** 문자열 매개 변수에 대한 속성이 있으며 사용자는 매개 변수에 여러 줄의 텍스트를 입력할 수 있습니다.
* **빈 값 허:** 문자열 매개 변수는 텍스트가 동적 데이터 필터링을 위해 판단을 위해 사용할 수 있는 빈 문자열 "" 일 수 있음을 나타내는 속성을 가지고 있습니다.
* **사용 가능한 데이터:** 매개 변수 드롭다운 목록을 구현할 때 사용됩니다. 사용 가능한 데이터는 드롭다운 목록의 데이터에 사용되며 데이터 집합에서 가져오거나 수동으로 추가할 수 있습니다. 데이터 집합에서 가져오려면 데이터 집합, 데이터 필드 및 레이블 필드를 설정해야 합니다. 여기서 데이터 필드의 값은 데이터 집합 SQL 문에 전달된 실제 매개 변수 값으로 사용됩니다. 레이블 필드의 값은 드롭다운 목록 상자에 표시되는 문자열입니다. 다음 그림은 사용 가능한 데이터의 적용 효과입니다.
* **기본 값:** 보고서 매개변수의 기본값을 설정합니다. 기본값은 데이터 집합의 데이터 필드에서 가져오거나 수동으로 추가할 수 있습니다. 보고서 매개 변수에 대한 기본 데이터를 설정한 경우 보고서를 처음 열 때 쿼리 결과를 직접 표시하기 위해 사용자 입력이 필요하지 않습니다.

![](https://help.grapecity.com.cn/download/thumbnails/56537582/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1615884259000\&api=v2) 보고서 매개 변수가 추가되면 매개 변수 후 쿼리 매개 변수 삭제를 클릭하고 쿼리 매개 변수가 많은 경우 쿼리 매개 변수의 오른쪽에 있는 위쪽 및 아래쪽 화살표를 클릭하여 쿼리 매개 변수가 표시되는 순서를 조정할 수 있습니다.![](https://help.grapecity.com.cn/download/thumbnails/56537582/image2021-3-16_16-44-46.png?version=1\&modificationDate=1615884286000\&api=v2)

![](https://help.grapecity.com.cn/download/thumbnails/56537582/%E6%AD%A5%E9%AA%A44.png?version=1\&modificationDate=1615884310000\&api=v2)  보고서를 미리 보면 매개 변수 입력 상자가 나타납니다. 그러나 현재 실제 데이터 필터링에는 보고서 매개 변수가 사용되지 않습니다.

데이터 필터링에 보고서 매개 변수를 사용하거나 보고서 매개 변수를 보고서로 직접 드래그하여 매개 변수 값을 직접 표시할 수 있습니다.
