# 리스트뷰 로딩 애니메이션

양식 데이터가 크거나 네트워크 속도가 너무 느린 경우 양식 데이터를 처음 로드할 때 기본 로딩 애니메이션이 있을 수 있습니다.![](https://help.grapecity.com.cn/download/thumbnails/80959225/image2023-2-6_16-13-47.png?version=1\&modificationDate=1675671227000\&api=v2), 이 애니메이션을 변경할 수 있습니다.



리본 메뉴에서 "파일-설정-응용 프로그램 설정"을 선택하면 테이블 로딩 애니메이션을 볼 수 있습니다.

<figure><img src="../../.gitbook/assets/image (722).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (1425).png" alt=""><figcaption></figcaption></figure>

![](https://help.grapecity.com.cn/download/thumbnails/80959225/image2023-2-6_16-39-10.png?version=1\&modificationDate=1675672750000\&api=v2) 버튼 클릭 시,  리스트뷰 로딩 애니메이션 대화 상자가 나타납니다.

* CSS 애니메이션: 내장된 로딩 애니메이션을 선택할 수 있으며 애니메이션 색상도 설정할 수 있습니다.

<figure><img src="../../.gitbook/assets/image (1918).png" alt=""><figcaption></figcaption></figure>

* GIF 애니메이션: 로컬 GIF 사진을 선택하거나 사진의 URL 주소를 입력할 수 있습니다. 오른쪽에서 GIF의 정적 효과를 미리 봅니다
