# 인증 모드

사용자가 웹 사이트의 페이지를 방문할 때 일반 인증, Windows 도메인 인증 및 외부 서비스 인증 세 가지 인증 모드가 있습니다.

리본 메뉴 모음에서 보안-> 인증 모드를 선택하고 Forguncy 인증, Windows  인증 및 외부 서비스 인증을 선택합니다.

![](<../../../.gitbook/assets/image (577).png>)

### Forguncy 인증&#x20;

포건시를 사용하는 사용자 관리 서비스는 사용자를 인증합니다. 사용자 로그인 페이지를 표시하려면 사용자 이름과 암호를 입력하여 로그인해야 합니다.

사용자를 만들 때 사용자 만들기를 참조하여 사용자를 인증하려면 Forms를 선택합니다.

### Windows 인증&#x20;

회사 서버에서 Active Directory의 사용자 관리를 사용하는 경우 회사의 Windows 도메인 계정에 로그인한 다음 포건시 앱에 자동으로 로그인할 수 있습니다.

이 인증 모드는 사용자 로그인 페이지를 표시하지 않지만 사용자 관리 웹 사이트에서 사용자를 미리 만들어야 하며 사용자 만들기를 위해 "Windows 인증 사용자"를 선택해야 합니다.

### 외부 서비스 인증&#x20;

일부 웹 사이트는 Google, Microsoft, WeChat 등과 같은 OpenId(Oauth2) 인증 서비스를 제공합니다. 포건시 사용자가 응용 프로그램에 액세스하면 포건시 페이지가 타사 웹 사이트 로그인 페이지로 리디렉션되고 사용자가 타사 웹 사이트에 로그인할 수 있으며 브라우저가 자동으로 포건시로 다시 이동하고 자동으로 로그인합니다. 타사 웹 사이트에 이미 로그인한 경우 포건시 응용 프로그램은 타사 웹 사이트의 동일한 계정을 통해 포건 응용 프로그램에 자동으로 로그인합니다.
