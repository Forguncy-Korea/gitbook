# 기본유형

## 기본유형

서버단 명령을 만들 때 기본 형식 및 배열 형식을 포함 하는 형식의 파라미터를 설정할 수 있습니다. 이 섹션에서는 문자열, 숫자, 날짜 또는 기타 간단한 형식의 데이터와 같은 기본 형식의 파라미터에 대해 설명합니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FNHoExsqCpBzdSkpU5FX7%252Fimage.png%3Falt%3Dmedia%26token%3D49fd8021-b056-49f2-9a4f-c204fe22fdff\&width=768\&dpr=4\&quality=100\&sign=f8dc3b0b2a3deaa6690e7804509d8407de19712cdd9ed858463a80a316bde5cb)

### 기본형식의 파라미터 <a href="#undefined" id="undefined"></a>

서버단 명령을 만들 때 서버단 명령 편집 대화 상자에서 파라마터 탭을 선택하고 파라미터 추가를 클릭한 다음 기본 매개 변수 유형을 기반으로 파라미터 이름 및 파라미터 유형을 설정할 수 있습니다.

예를 들어 직원을 추가하는 서버단 명령을 만듭니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2F%25E6%25AD%25A5%25E9%25AA%25A41.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=3f6d091a097bc277e42a491ef1dbe920281db260ba48172b1c4e0b5a5936e4f8) "직원 추가"라는 이름의 서버단 명령을 만든 다음 파라미터를 설정하고 다음 그림과 같이 3가지 기본 형식의 파라미터를 추가합니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FZJVIK2zutIrwZ7fmjXK7%252Fimage.png%3Falt%3Dmedia%26token%3Dfd43e785-ad6e-4f84-94bf-c5daacb0ac87\&width=768\&dpr=4\&quality=100\&sign=0cf2ff740f1bdfb8dd33e72bcf6e9cbd5585d460c1298577b19e8b3fae5d221d)

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2F%25E6%25AD%25A5%25E9%25AA%25A42.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=d0588b536975e98822c1a679330741cec5558a230c6a1532a35f4be8fe6ef67d) 파라미터에 대한 데이터 유효성 검사를 설정합니다. 다음 그림과 같이 파라미터 뒤에 있는 데이터 유효성 검사 아이콘을 클릭하여 팝업 데이터 유효성 검사 대화 상자에서 설정합니다.![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2Fimage2020-9-17_14-33-36.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=bb7d9e2ec167ce38a2d0e171d537335cc1ac074372f0144f73b9a6421e2beb23)

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FmTAeJVrcZRme2QK2qleE%252Fimage.png%3Falt%3Dmedia%26token%3D010173f4-b5ca-4292-aabf-2da6b31aa84a\&width=768\&dpr=4\&quality=100\&sign=6b03d4c38aee0441cb386ec75c22969addd265ea0874970421886857b30ff867)

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2F%25E6%25AD%25A5%25E9%25AA%25A43.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=5c1bf07c81954348c949dc797c212a8973ffb7234c32fc22d005b95a44f9c689) 파라미터를 설정한 후 명령을 편집하고 데이터 테이블 작업 명령을 선택합니다. 작업 유형은 추가, 필드 추가, 필드 값 설정에 대한 해당 매개 변수입니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FciI5hQfrCkW6DJPH7Ycg%252Fimage.png%3Falt%3Dmedia%26token%3Da10bd600-2d09-47a1-8009-4ca33316fbd1\&width=768\&dpr=4\&quality=100\&sign=b67e30fbfd66272c1d6f7aea7029df092024636ea88bdca806f61678b5b17008)

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2F%25E6%25AD%25A5%25E9%25AA%25A44.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=08ec5b0af8865847eb69ede971e3a81f720c3128a9f432c11cf7ef393d228456) 설정이 완료되면 이 서버단 명령을 호출할 수 있습니다.

예를 들어 페이지에서 셀 범위를 선택하고 버튼을 설정합니다. 버튼의 명령을 편집하고 "서버단 명령 호출"으로 명령을 선택한 다음 서버단 명령 뒤에 있는 드롭다운을 클릭하고 드롭다운 목록에서 "직원 추가"라는 서버단 명령을 선택합니다.

"직원 추가" 서비스 쪽 명령에 파라미터가 있기 때문에 모든 파라미터가 자동으로 나열되므로 파라미터를 설정해야 하며 파라미터는 페이지의 해당 셀입니다. 그런 다음 "호출이 성공한 후 데이터 다시 로드"를 선택하여 데이터를 추가한 후 페이지가 새로 고쳐지고 추가된 데이터가 표시됩니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FLM8QYL9lLfXpf58ekcxM%252Fimage.png%3Falt%3Dmedia%26token%3D1c953125-8060-4477-9571-6db7263ecc0b\&width=768\&dpr=4\&quality=100\&sign=ea244fa105cb0ef916bb2f0d4e28b973e868e226a87475854e32a0a834fe94fb)

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2Fhelp.grapecity.com.cn%2Fdownload%2Fthumbnails%2F72357211%2F%25E6%25AD%25A5%25E9%25AA%25A45.png%3Fversion%3D1%26modificationDate%3D1648092603000%26api%3Dv2\&width=300\&dpr=4\&quality=100\&sign=4d7370f8d590351dd81815313b7557778ef70fe3d64d92ddeb2e16a2f03f73a1) 실행 후 페이지의 주문날짜 텍스트상자 외 값을 입력한 후 직원 추가 버튼을 클릭하면 매개 변수의 데이터 유효성 검사에 설정된 오류 경고가 표시되는 경고 상자가 나타납니다.

![](https://doc.forguncy.co.kr/~gitbook/image?url=https%3A%2F%2F1191936652-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FMDaEsmK9OwD7QYBNk9Sn%252Fuploads%252FCaM9S03or3nax5PnlGMh%252Fimage.png%3Falt%3Dmedia%26token%3Db59c900b-8678-4dfd-9263-50da1588a8af\&width=768\&dpr=4\&quality=100\&sign=b01b0d195121dffc987807b72c8a69ec13373f5a53a13416ea50a21dcbdc0c40)

주문날짜 텍스트 상자에 값을 입력한 다음 직원 추가 버튼을 클릭하면 페이지에 설정된 값이 데이터 테이블에 데이터를 추가하는 데이터 테이블 작업 명령을 실행하는 서비스 쪽으로 전달됩니다.
