# 여러개 그룹화 및 통계

다단계 그룹화는 각 "지역" 내에서 "지방"과 같은 기존 그룹 내에 그룹화를 추가합니다.

다단계 그룹화를 추가하는 방법은 기본적으로 단일 수준 그룹화를 추가하는 것과 동일하며, 이 경우 단계를 보여 줍니다.

![](https://help.grapecity.com.cn/download/thumbnails/64455840/%E6%AD%A5%E9%AA%A41.png?version=1\&modificationDate=1633694701000\&api=v2)  보조 그룹화를 추가합니다. 성별 판매 주 필드를 테이블 그룹화 창으로 끌어 보조 그룹을 추가합니다. 이 시점에서 첫 번째 수준 그룹화 내에서 세부 정보 행의 위쪽과 아래쪽에 두 개의 테이블 행이 더 나타나고 두 번째 수준 그룹화의 헤더와 그룹화된 끝이 표시됩니다.

![](<../../../.gitbook/assets/image (2079).png>)

그룹화 창에서 그룹 이름을 클릭하면 해당 데이터 범위가 검은색 상자로 표시됩니다. 표시되는 테이블의 첫 번째 수준 그룹화 범위는 두 번째 수준 그룹보다 크며 첫 번째 수준 그룹화의 데이터 범위에는 두 번째 수준 그룹화가 포함됩니다.

![](https://help.grapecity.com.cn/download/thumbnails/64455840/%E6%AD%A5%E9%AA%A42.png?version=1\&modificationDate=1633694701000\&api=v2)  데이터를 바인딩합니다. 보조 그룹 헤더는 그룹화 필드 "성"에 자동으로 바인딩됩니다. 다음 그림과 같이 두 번째 수준의 그룹화 끝도 주문 번호 필드에 바인딩됩니다(계산은 자동으로 수행됨).

![](<../../../.gitbook/assets/image (1677).png>)

![](https://help.grapecity.com.cn/download/thumbnails/64455840/%E6%AD%A5%E9%AA%A43.png?version=1\&modificationDate=1633694701000\&api=v2)  보고서를 미리 봅니다. 그림에서 볼 수 있듯이 표시되는 데이터는 그룹화된 분할에 따라 표시되며, 동일한 필드 "완료여"가 바인딩되어 있지만 각 데이터 범위가 다르기 때문에 최종 통계 결과가 다릅니다.

두 번째 수준의 그룹화의 끝은 두 번째 수준의 그룹화 데이터 범위의 "성"에 대한 데이터를 계산합니다.
