# 목표값 찾기(GoalSeek)  수식

Excel에서 1개 셀의 데이터를 변경하여 계산식이 있는 셀이 원하는 값이 되도록 셀의 데이터를 변경하여 원하는 값을 찾는 목표값 찾기 기능이 있습니다. 포건시(Forguncy)에서도 목표값 찾기 수식을 사용할 수 있습니다.

> 목표값찾기 기능은 서버단 명령에서 사용이 불가능합니다.

### 사용방법

<figure><img src="../../../.gitbook/assets/image (2607).png" alt=""><figcaption></figcaption></figure>

* Parameter Name:  수식에서 사용되는 파라미터, 파라미터의 이름과 값을 설정합니다. 값은 셀 값이나 수식 계산에서 가져올 수 있습니다.
* Variable: 값을 바꿀   셀
* Formula: 목표값 찾기에 사용되는 수식
* Goal Value: 이 수식의 목표값, 찾는 값
* Set Solved Value To: 해결된 값을 포건시 명령 시스템의 파라미터에 저장합니다. 이 값은 다른 명령에서도 사용이 가능합니다.
